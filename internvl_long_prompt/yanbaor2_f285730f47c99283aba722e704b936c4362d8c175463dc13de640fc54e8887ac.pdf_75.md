# 2020科技、传媒和电信行业预测

广播公司大部分收入来自广告商，广告商支付钱在广播公司的节目上投放商业广告。广播公司的部分收入也来自发行商，发行商向广播公司支付钱将广播公司的节目打包销售。大多数广播公司并不直接从观众那里获得收入，而是执行几乎全部收入将购买的频道打包销售。发行商向观众绑定出售频道，月费约为110美元。总体而言，发行商不会向“广告商”出售广告位。

在企业层面，广播公司和分销商之间的界限一直在逐渐模糊，起初的发行商收购了广播公司，反之亦然。事实上，今天世界上大多数的电视业从业者并不只参与单个领域：他们既是广播商又是发行商。

本章还使用术语“传统电视”来指代天线电视，有线电视、电信IPTV、卫星电视和MVVPD。传统电视包括广播公司提供的所有免费和付费性电视，无论是通过MVVPD还是天线传送，而不包括Netflix和Amazon Prime等服务，这些服务归为订阅视频点播（SVOD）服务，或者用户产生内容（UGC）服务，如YouTube和UGC在很多方面都与电视效果相似，观众经常使用SVOD或UGC替代传统电视，但SVOD或UGC与传统电视仍存在较大差异，不会将SVOD或UGC归为传统电视行业。

## 数据反映天线电视市场规模

据我们所知，全球天线电视市场规模数据从未公布。各个国家存在数据，监管机构记录了从模拟地面电视广播到数字电视广播的转变方式（与重新分配频谱信息相关），但似乎还没有一项全球研究对使用天线电视的人数进行调查。

基于83个国家/地区、66亿总人口的真实数据，预计2020年天线电视使用人数将达到16亿。
