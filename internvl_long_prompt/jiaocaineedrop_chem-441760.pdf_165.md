# FUNCTIONAL GROUPS IN ORGANIC CHEMISTRY

Functional groups are the characteristic groups in organic molecules that give them their reactivity. In the formula below, R represents the rest of the molecule and X represents any halogen atom.

- Hydrocarbons
- Halogen-containing groups
- Oxygen-containing groups
- Nitrogen-containing groups
- Sulfur-containing groups
- Phosphorus-containing groups
