# 第十课 极地地区

## 冰雪覆盖的地区

### (1)南极与北极地区的比较：

| 地理位置 | 南极地区 | 北极地区 |
| --- | --- | --- |
| 地理位置 | 地球最南端 | 地球最北端 |
| 范围 | 南极圈(66.5°S)以南的广大区域 | 北极圈(66.5°N)以北的广大区域 |
| 海陆状况 | 中间是南极大陆，周围被海洋环绕 | 中间是北冰洋，周围是亚欧、北美洲的陆地和岛屿 |
| 气候状况 | 寒冷、干燥、烈风 | 寒冷、降水量极少、风速较大 |
| 资源状况 | 矿产、水产、海洋生物、旅游 | 主要是矿产资源 |
| 居民 | 无人 | 因纽特人(北美洲)、拉普人(欧洲) |
| 代表动物 | 企鹅 | 北极熊 |
| 重要性 | 控制亚洲、欧洲、北美洲 | 控制太平洋、大西洋、印度洋 |
