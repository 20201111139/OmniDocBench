# 筑记十九 走向世界的中国

## 促进区域的可持续发展

### (1) 基本国情

①国土辽阔，区域差异大：
- 因人口众多，国土面积广大，地形地貌复杂，自然条件差异大。

②区域差异：
- 可持续发展，因地制宜发展经济。

③区域合作：加强区域联系，互惠互利。

④北方保护林，东起黑龙江的安县，西至新疆的乌鲁木齐山，包括13个省区的西北、华北、东北地区，被誉为“世界生态工程之最”。
