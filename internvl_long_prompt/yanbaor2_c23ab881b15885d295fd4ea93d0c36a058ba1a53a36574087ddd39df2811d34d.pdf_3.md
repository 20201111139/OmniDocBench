# 新一代无线技术加速企业创新与转型

## 过去
去三十多年里，无线互联网连接经历了翻天覆地的变化，人们对高速连接的期望，以及人与物、物与物的海量连接需求持续推动着无线技术的变革。如今，随着5G和Wi-Fi 6等新一代无线技术的完善和广泛应用，数十亿智能家居传感器、工业设备以及人工智能计算机将接入互联网。物联网网络端指标增长呈现指数级增长，未来互联网物联网网络承载的要求达到了前所未有的高度，无线网络技术投资布局已经成为全球企业的关键发展目标。

## 现在
2020年新冠疫情肆虐全球，全球企业经营受到冲击。为了了解危机期间全球企业的先进无线技术采用情况，德勤针对全球的网络技术高管展开调研。在这项调研中我们着重研究了三个极块，受访的企业在受疫情影响后对先进无线技术态度的转变、先进无线技术对企业创新和转型带来的助力、以及技术落地所需的生态环境。在调研过程中，全球网络技术决策者明显表现出对于先进无线技术的重视，他们都认为5G和Wi-Fi 6将成为商业成功的关键助力。新冠疫情期间，提高网络连接以支持远程办公、在线学习和自动化的需求激增，更生动证明了企业对安全、优质、无处不在的网络连接的诉求。

## 未来
可以肯定的是，先进无线技术将以惊人的速度，以及低功耗、高带宽和低时延特性，帮助企业提升灵活性，应对突如其来的业务中断局面，催生创新解决方案并推进企业执行创新用例，为社会各个领域带来如其业务的局面，催生创新解决方案并推进企业执行创新用例，为社会各领域带来如其业务的局面，催生创新解决方案并推进企业执创新用例，为社会各领域带来如其业务的局面，催生创新解方案并推进企业执行创新用例，为社会各领域带来如其业务的生态环境。作为物联网发展基石的先进无线技术，将结合人工智能、大数据和区块链等技术，推进人类社会从“信息化”迈向“智能化”时代。

## 5G和Wi-Fi 6的同频采用趋势也很明显
作为最积极探索和采用先进无线技术的国家之一，中国的5G采用率高达70%，Wi-Fi 6部署及试点比例（72%）也处于最高水平。2020年5月，国家提出加快5G网络、数据中心等新型基础设施建设进度。在各方支持下，2020年底中国5G基站总量超过78.8万个，实现所有地级以上城市5G网络全覆盖。2020年同期，中国WLAN市场总体规模达到8.7亿美元，Wi-Fi 6占总体WLAN市场31.2%，规模达2.7亿美元。“二者对于无线技术的未来都至关重要，在未来很长一段时间内，5G和Wi-Fi 6将会相互补充，共同推动无线技术的发展。”
