### 试题

1. 设定积分 \( \int_{0}^{1} \sin^2 x + \cos^2 x - 5 \, dx \).

   [解析]: 解 \( f(x) = \sin^2 x + \cos^2 x - 5 \), 则 \( df = 2 \sin x \cos x \, dx \), \( \sin^2 x + \cos^2 x = 2 \sin^2 x + 1 \), 则
   \[
   \int_{0}^{1} \sin^2 x + \cos^2 x \, dx = \int_{0}^{1} 2 \sin^2 x + 1 \, dx = \ln 2
   \]

2. 计算二重积分 \( \int_{D} \sin \left( \max \{ x, y \} \right) \, dy \, dx \)，其中区域 \( D = \{ (x, y) \mid 0 \leq x \leq y \leq \sqrt{y} \} \).

   [解析]: 解不妨设 \( D = \{ (x, y) \mid 0 \leq x \geq y \leq \sqrt{y} \} \)，由对称性可知
   \[
   \int_{D} \sin \left( \max \{ x, y \}^2 \right) \, dy \, dx = 2 \int_{0}^{1} \sin^2 x \, dx = 2 \int_{0}^{1} \sin x \, dx = 2
   \]

3. 求级数 \( \sum_{n=1}^{n} \frac{(-1)^{n-1}}{n(2n+1)} \) 的和。

   [解析]: 解考虑
   \[
   \sum_{n=1}^{n} \frac{(-1)^{n-2}}{n(2n+1)} + \sum_{n=1}^{n} \frac{(-1)^{n-3}}{n(2n+1)} = \frac{1}{2}
   \]
