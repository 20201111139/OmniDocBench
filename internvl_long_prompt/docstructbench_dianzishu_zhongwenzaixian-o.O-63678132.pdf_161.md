# 发展临沧

## 大美临沧系列丛书

### 加快建设“绿色银行”

伴随奋进的脚步和激越的鼓点，第十三个五年规划的建议，明确了临沧走过满载收获的“十二五”，开启了新的征程。“十三五”时期是临沧经济社会发展的指导思想和目标要求。中国共产党临沧市第四次代表大会批准孙青友同志代表三届市委所做的报告。大会充分肯定了三届市委的工作，认为市第三次党代会以来的五年，是抢抓机遇、重点突破、不断夯实发展基础的五年，是直面矛盾、迎难而上、不断赢得发展主动权的五年，是找准定位、站新历史起点、发挥优势、不断开创发展新局面的五年。

中国共产党成立95年来，党的光辉照边疆，临沧人民心向党。临沧各族人民的命运始终与党的命运休戚
