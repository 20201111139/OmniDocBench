### 评级说明
#### 1. 投资建议的比较标准
投资评级分为股票评级和行业评级

以报告发布后的6个月内的市场表现为比较标准，报告发布日后的6个月内公司股票（或行业指数）的涨跌幅相对同期的沪深300指数的涨跌幅为基准

#### 2. 投资建议的评级标准
报告发布日后的6个月内公司股票（或行业指数）的涨跌幅相对于同期的沪深300指数的涨跌幅为基准

### 股票投资评级
| 评级 | 说明 |
| --- | --- |
| 买入 | 相对沪深300指数涨幅15%以上 |
| 增持 | 相对沪深300指数涨幅5%-15%之间 |
| 中性 | 相对沪深300指数涨幅-5%~5%之间 |
| 减持 | 相对沪深300指数涨幅-5%以下 |

### 行业投资评级
| 评级 | 说明 |
| --- | --- |

| 看淡 | 行业低于整体市场表现 |
