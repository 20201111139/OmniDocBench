# 笔记二 陆地和海洋

大洲和大洋

1．地球 木球

（1）地球表面陆地面积占29％，海洋面积占71％，概括为“七分海洋，三分陆地”。

（2）分布：陆地主要集中在北半球，海洋主要集中在南半球.

2．土大洲和四大洋

（1）关于海洋、陆地的基本概念。


| 地域 | 概念 |
| -- | -- |
| 大洲 | 大陆和它附近的鸟屿 |
| 大陆 | 面积广大的陆地 |
| 岛屿 | 面积较小的陆地 |
| 洋 | 海洋的中心部分 |
| 海 | 是洋的部分，位于大洋边缘，面积较小.靠近大陆 |
| 海峡 | 沟通两个海洋之间的狭窄木 |


（2）海陆分布大势：

①六块大陆：亚欧大陆、非洲大陆、南极大陆、澳大到亚大陆、北美大陆、南美大陆

②七大洲：亚洲、非洲、北美洲、南美洲、南极洲.欧洲·大洋洲

③四大洋：太平洋、大西洋、印度洋、北冰洋

④大洲分界线：

亚.欧：乌拉尔山脉→乌拉尔河→里海→大高加索山脉→黑

海→土耳其海峡（沟通黑海和地中海）

亚非：苏伊士运河（沟通地中海和红海）

南，北美：巴拿马望河（沟遏太平洋和大西洋）。 亚、北美、白令海峡.
