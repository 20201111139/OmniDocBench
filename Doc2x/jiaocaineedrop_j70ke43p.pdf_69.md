## 1. 三角恒等变换

万能置换公式: \( \sin x = \frac{2\tan \frac{x}{2}}{1 + {\tan }^{2}\frac{x}{2}};\cos x = \frac{1 - {\tan }^{2}\frac{x}{2}}{1 + {\tan }^{2}\frac{x}{2}};\tan x = \frac{2\tan \frac{x}{2}}{1 - {\tan }^{2}\frac{x}{2}} \) ;

## (1)积化和差公式

\( \sin A\cos B = \frac{1}{2}\left\lbrack  {\sin \left( {A + B}\right)  + \sin \left( {A - B}\right) }\right\rbrack  ;\cos A\sin B = \frac{1}{2}\left\lbrack  {\sin \left( {A + B}\right)  - \sin \left( {A - B}\right) }\right\rbrack \) ;

\( \cos A\cos B = \frac{1}{2}\left\lbrack  {\cos \left( {A + B}\right)  + \cos \left( {A - B}\right) }\right\rbrack  ;\sin A\sin B = \frac{1}{2}\left\lbrack  {\cos \left( {A - B}\right)  - \cos \left( {A + B}\right) }\right\rbrack \) ;

## (2)和差化积公式

\[
\sin A + \sin B = 2\sin \frac{A + B}{2}\cos \frac{A - B}{2};\;\sin A - \sin B = 2\cos \frac{A + B}{2}\sin \frac{A - B}{2}\text{;}
\]

\[
\cos A + \cos B = 2\cos \frac{A + B}{2}\cos \frac{A - B}{2};\cos A - \cos B =  - 2\sin \frac{A + B}{2}\sin \frac{A - B}{2}\text{;}
\]
