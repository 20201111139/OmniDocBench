7. 如图,梯形ABCD中, \(AB||CD\). 若\(AB=1\), \(CD=2\), \(AC=6\), \(S_{\triangle AOB}=1\). 则\(AO=\)______
\(S_{梯形ABCD}=\)______

解: \(\because AB||CD\)

\(\therefore \triangle AOB \backsim \triangle COD\)

则 \(\frac{AO}{CO} = \frac{AB}{CD}\) \(\because AB=1, CD=2, AC=6\)

\(\therefore \frac{AO}{CO} = \frac{1}{2} \Rightarrow CO = 2AO\)

又\(\because AC = AO + CO = 3AO \Rightarrow 3AO = 6\), \(AO=2\)

又\(\because \frac{S_{\triangle AOB}}{S_{\triangle COD}} = (\frac{AB}{CD})^2 = (\frac{1}{2})^2 = \frac{1}{4} = \frac{1}{4} S_{\triangle COD}\)

\(\therefore S_{\triangle COD} = 4\)

\(\because S_{\triangle AOB} 与 S_{\triangle BOC} (不同底, 同高)\)

且底之比为 \(OA:OC = 1:2\)

\(\therefore S_{\triangle AOB} : S_{\triangle BOC} = 1:2\) \(\because S_{\triangle AOB} = 1\)

\(\therefore S_{\triangle BOC} = 2\). 同理: \(S_{\triangle AOD} 与 S_{\triangle COD} (不同底, 同高)\)

且底之比为 \(OA:OC = 1:2\) \(\therefore S_{\triangle AOD} : S_{\triangle COD} = 1:2\)
