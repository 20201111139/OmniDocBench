# 运用各种描写手法来塑造人物形象

体现包括人物的容貌: 身材 服饰等_ 它对于人物性格和人物形象的完整性。肖像描写可以迅速掌握人物的外在特征，从正面刻画人物肖像。

# 正面描写

嫂的眼神的三次播写。

特征个性化的对话; 都极直性格。

# 语言描写

杨二嫂的尖刻利己，阿@的自欺自慰，无不绘形绘色，跃然纸上，富有特征的内心独白，也是透视人物内心隐秘。

财产有可能失去时，情不自禁地对自己说道: 那简直是抹自己的脖子! 它直接揭示了人物对金钱赤裸裸的占有欲。

# 动作描写

的外在显现，如中就表现了林黛玉的小心谨慎，推动情节发展，兴奋等，表现人物思想品质，刻画人物性格。

# 心理描写

可以使人物在人物描写中具有重夔的意义，如《老人与海》中对桑地亚哥的心态描写：“但愿我没有钓到这条鱼。”

动摇地表现出桑地亚哥绝不放弃每次机会，即使失败宁，同样坦荡荡，不失重压下的优雅风度。

# 细节描写

林黛玉小心谨慎的性格。

# 侧面描写

侧面描写又叫间接描写，是从侧面烘托人物形象。通过对周围人物或环境的描绘来表现所要描写的对象，以使其鲜明突出。
