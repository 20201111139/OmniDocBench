# 例题讲解

括号里最大能填几? 徐老师去商店买笔每支 3 元,他带了 20元可以买几支? 还剩几元?

# 思路分析:

这是一道有余数的除法题. 可以买 48个苹果, 平均分给 9个同学, 每人分几个?

48:9 = 5 (个), 余数 3 (个)

读作: 9义11<48

# 自我解答:

三数学诊断。(对的打"~/ ,错的打"义" ,并改正 )

|4 改正:|5 改正:|
|---|---|
|3.1|3)1|
|1 2|1 5|
|5|3|
