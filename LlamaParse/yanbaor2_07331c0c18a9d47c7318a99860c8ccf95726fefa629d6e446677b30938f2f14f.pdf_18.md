# 中国新消费明日之星

# 第二章 聚焦; 细分行业洞察与赛迸聚集

# 品牌介绍 -茶百道

# 近期表现亮点

茶百道于2008年创立, 13年间, 茶百道快速发展并不断扩张门店规模, 近年来增速更为明显, 2018年起, 为快速拓展门店; 从单一的直营模式转为 "直营+加盟" 模式, 扩店速度迅猛, 2020年12月全囤门店数己逾3,000家, 而截至2021年9月, 茶百道在全国的门店数量己超过5,000家。除了四家直营店, 其余均为加盟, 己覆盖国内西南华北、东北、西北、中南、华东等区域的逾250个城市。

# 品牌展望

茶百道通过推出中国特色茶饮与熊猫成功塑造品牌, 同时在定价模式上补足了新式茶饮业中端市场的空白。在蜜雪冰城霸占10元以下低消市场, 奈雪的茶、喜茶等抢占20元以上高端赛道的情况下, 腰部消费区尚未出现统治力量, 我们认为茶百道强势入局, 有望跻身腰部前列。

# 品牌形象定位有记忆点

为突出中式茶饮形象。在外观设计上融入了漫画风的国宝大熊猫的元素, 提高了品牌辨识度。更能抓住眼球。价格定位补充了市场空缺。在高端和低端市场分别己有喜茶、蜜雪冰城等品牌的情况下, 将产品定价在新式茶饮的腰部, 单价基本在16元-25元之间, 以差异化价格获取客户。

# 精简菜单, 突出爆品

茶百道将莱单从逾40款缩至25款, 并通过对市场潮流的追踪, 及时迭代菜单, 成功打造了在消费者中朗朗上口的豆乳玉麒麟等系列爆品。茶百道通过线上内容投放与用户共创点单攻略等新营销战略, 加强与消费者的联系。茶百道与小红书、抖音等各路美食博主共建点单攻略, 也会通过本地美食公众号推广; 在广覆盖结合区域化宣传的强宣发下抓住消费者小确幸, 小生活, 小意外的消费心理, 增强品牌的影响力。
