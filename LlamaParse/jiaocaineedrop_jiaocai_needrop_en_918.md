# 醉翁亭记

# 教学目标

掌握初二必背文言文《醉翁亭记》中的重点实词。虚词和旬式。

感受作者与民同乐的心境。

# 走进作者

欧阳修(1007年~1072年),字永叔, 自号醉翁,北宋文学家,政治家。史学家和诗人，"唐宋八大家"之一。因家中有一万卷、一千卷金石遗文;一张琴一局棋一壶酒。一老翁。晚年又号六一居士。

# 文体简介

记: 一种文体。可以写景、叙事，多为议论。但目的是为了抒发作者的情怀和抱负(阐述作者的某些观念)。

# 醉翁亭记

欧阳修(北朱)

山行六七里。渐闻水声潺潺而泻出于两峰之间者，酿泉也。峰回路转，有亭翼然临于泉上者，醉翁亭也。

作亭者谁? 山之僧智仙也。名之者谁? 太守自谓也。太守与客来饮于此。饮少辄醉。而年又最高。故自号曰: 醉翁也。醉翁之意不在酒，在乎山水之间也。山水之乐，得之心而寓之酒也。

野芳发而幽香，佳木秀而繁阴，风箱高洁，水落而石出者，山间之四时也。朝而往，暮而归，四时之崇不同。而乐亦无穷也。

至于负者歌于途，行者休于树，前者呼，后者应，伛偻提携，往来而不绝者，滁人游也。临溪而渔，溪深而鱼肥。酿泉为酒，泉香而酒冽，山肴野蔌，杂然丽前陈者，太守宴也。宴酣之乐，非丝非竹，射者中，弈者胜，觥筹交错，起坐而喧哗者，众宾欢也。苍颜自发，颓然乎其间者，太守醉也。
