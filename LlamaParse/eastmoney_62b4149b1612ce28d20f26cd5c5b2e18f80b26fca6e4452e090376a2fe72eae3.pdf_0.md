# 行情走势图

常熟银行 沪深300指数

息差小幅收窄,看好微贷持续复苏。公司上半年年化净息差3.00%,保持相对稳定,我们按期初期末余额测算2季度季净息差较1季度环比收窄158基点至2.78%,背后可以看到资负两端均有拖累,2季度生息资产收益率环比下降98基点至4.94%,计息负债收益率环比抬升78基点至2.34%,预计主要与贷款利率的持续下行和存款定期化趋势的延续有关。规模方面,上半年末公司总资产较年初增长10.4%同比增速(15.6%)较1季度(18.0%)有所放缓;其中贷款较年初增长11.0%,存款较年初增长13.5%。不过从信贷增量结构来看2季度零售端投放力度加大,贡献了71%的信贷增量,其中个人经营性贷款投放为主要驱动因素,较年初增长14.0%。
