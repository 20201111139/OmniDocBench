1. 求定积分 $\int_0^\ln 2 \frac{\sin x \cos x}{\sin^3 x + \cos^3 x - 5} \, dx$.

【答案】：令 $t = \sin^2 x$，则 $dt = 2 \sin x \cos x \, dx$，$\sin^4 x + \cos^4 x = 2^2 - 2t + 1$，则

$$\int_0^\ln 2 \frac{\sin x \cos x}{\sin^3 x + \cos^3 x - 5} \, dx = \frac{1}{12} \int_0^1 \left( 1 - \frac{1}{t + 1} \right) \, dt = \ln 2 - \frac{1}{12}$$

【解析】：

2. 计算二重积分 $\iint_D \sin (max \{x^2, y^2\}) \, dx \, dy$，其中区域 $D = \{(x, y) \mid 0 \leq x, y \leq \sqrt{\pi} \}$.

【答案】：解不妨设 $D_1 = \{(x, y) \mid 0 \leq y \leq x \leq \sqrt{\pi} \}$，由对称性可知

$$\iint_D \sin (max \{x^2, y^2\}) \, dx \, dy = 2 \int_0^\sqrt{\pi} \sin x^2 \, dx \int_0^x \sin y^2 \, dy = 2 \int_0^\sqrt{\pi} x \sin x^2 \, dx = 2$$

3. 求级数 $\sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n(2n+1)}$ 的和。

【答案】：解考虑

$$\sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n(2n+1)} = \sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n} + 2 \sum_{n=1}^{\infty} \frac{(-1)^n}{2n+1}$$

注意到 $\ln(1 + x) = \sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n} x^n, x \in (-1, 1]$.

由幂级数相关性质可知

$$f'(x) = \sum_{n=1}^{\infty} (-1)^n x^{2n} = -\frac{x^2}{1 + x^2} = \frac{1}{1 + x^2} - 1, x \in (-1, 1).$$

5. 设 $f(x, y) = x^3 - 3x^2y - y^3 + x^2 - y$.

【答案】：证明 (1) 注意到 $f(0, 0) = 1, \frac{\partial f}{\partial y}(0, 0) = -3x^2 - 3y^2 - 1 \neq 0$. (2) 在 $x^3 - 3x^2y - y^3 + x^2 - y = 0$ 两边同时对 $x$ 求导，有

$$3x^2 - 6xy - 3x^2y - y^3 + 2x - y' = 0$$

代入 $x = 0$ 和 $y(0) = 0$，得到 $y'(0) = 0$. 重复上述操作，得到

$$6x - 6y - 12xy' - 3x^2y' - 6y(y')^2 - 3y^2y'' + 2y'' = 0.$$ 

代入 $x = 0$ 和 $y(0) = y'(0) = 0$，得到 $y''(0) = 2 > 0$.【解析】：
