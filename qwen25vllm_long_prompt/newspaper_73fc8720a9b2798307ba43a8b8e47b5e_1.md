# 更多信贷资金有望流向“县域”

本报记者 李海涛

银监会副主席蒋定之12月19日在“中国县域经济发展高层论坛”上表示，银监会将引导更多信贷资金流向县域经济，拓宽县域经济资金来源渠道，加大政策力度以提升县域金融服务水平。

县域经济是“头联农村、尾接城市”的国民经济重要组成部分，是推动农村经济转型、农民转移就业和农村经济持续健康发展的历史重任。一头连着城市，一头连着农村和人口，独占65%的农村劳动力。加快县域经济发展，无疑是破解城乡二元经济结构最具现实意义的战略选择，也是统筹城乡协调发展的重要举措。
