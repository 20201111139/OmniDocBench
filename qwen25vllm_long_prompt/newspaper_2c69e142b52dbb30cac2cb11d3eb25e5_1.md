# 土地是农民的一种财产权利

## 国务院发展研究中心副主任 韩俊

2010年1月至3季度,国家土地出让金收益是1.68万亿元。地方政府财政预算外收入里来自卖地的钱,估计得到40%,这些钱就是土地出让金。现在土地财政在很多地方,政府成了一个卖地的大公司,这是不正常的。政府如果像有些企业家的,成了经营土地的大公司,这就容易损害农民民生。因为他们拿走了以后,政府的民生投入上不去,但是基础设施建设上来了,基础设施建设就提高了城市的土地价值,土地价值提高了。作为政府,改善民生,提供必要的公共服务,这也是政府应尽的义务。但如果一个城市把房价搞这么高,下一步把蔬菜等一些基本生活消费品的成本也搞得很高,老百姓怨声载道,还有什么和谐可言?

从农村土地出让获得的收益却大部分都没有用在农业上

说到这里,来了这么久,这个钱干什么用了?一定要试问这个问题。钱到哪里去了,到很多地方搞建设,卖地卖了多少钱是不清楚的,但是如果说问卖地的钱干什么去了,很多地方不愿意告诉你。我看到一个最简单的完成,就是修路修这些钱,真正用于农业和农民的只有5%左右。剩下的钱是用于除路外的基础建设、开发区的建设,用于城市的发展。我们现在在建设土地,但是用之于城。在这个问题上,我们还是不是工业反哺农业,是不是城市支持农村?在这个问题上,其实这是个认识是不正确的。背后其实是一个利益和分配问题。世界上没有一个国家因为按照市场价值对农民的土地进行补偿,其发展就受到抑制了。所以应该让农民分享更多的土地的收益,要增加农民的收入,财产性收入等等。

现在总的来讲补偿标准在不断提高,但是在农民的眼里,这个补偿标准还是偏低

特别是某些重大基础设施的建设给农民的补偿就偏低。比如说高速公路,167公里的农民土地征用款只有公路造价的1%,就是我们国家这些高速公路,现在包括也包括高速铁路,里面的土地成本是非常低的,实际是压低了对农民的补偿标准换来了我们那些基础设施领域城市化的快速发展。

无论是工业还是农业,还是商业,给农民赔偿应该是一个刚性的,所谓公益性它就有一定的强制性,你不能说它是一个公益性的某个建设就让农民出钱,土地是农民的,土地是农民的,财产性权利,现在宪法也规定了,物权法也规定了,包括财产权包括财产权的财产权利,包括平等的保护,在征地问题上一定要树立这个理念。我们能够用我们的公权力去侵犯农民的私权利,应该成为一个基本的准则。

现在很多人为说,给农民补偿高了就会影响发展,其实这个认识是不正确的。背后其实是一个利益和分配问题。世界上没有一个国家因为按照市场价值对农民的土地进行补偿,其发展就受到抑制了。所以应该让农民分享更多的土地的收益,要增加农民的收入,财产性收入等等。
