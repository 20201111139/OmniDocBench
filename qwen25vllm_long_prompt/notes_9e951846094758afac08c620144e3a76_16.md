## 总结帝笔记—初三寒假班第三讲

7. 如图, 梯形ABCD中, AB∥CD. 若AB=1, CD=2, AC=6, S△AOB=1, 则AD=____, S梯形ABCD=____

![梯形ABCD](image.png)

解: ∵ AB∥CD

∴ △AOB∽△COD

∴ \(\frac{AO}{CO} = \frac{AB}{CD}\)

∵ AB=1, CD=2, AC=6

∴ \(\frac{AO}{CO} = \frac{1}{2} \Rightarrow CO = 2AO\)

又∵ AC=AO+CO=3AO \(\Rightarrow\) 3AO=6, AO=2

又∵ \(\frac{S_{\triangle AOB}}{S_{\triangle COD}} = \left(\frac{AB}{CD}\right)^2 = \left(\frac{1}{2}\right)^2 = \frac{1}{4} = \frac{1}{S_{\triangle COD}}\)

∴ \(S_{\triangle COD} = 4\)

∵ \(S_{\triangle AOB}\) 与 \(S_{\triangle BOC}\) (不同底, 同高)

且底之比为OA:OC=1:2

∴ \(S_{\triangle AOB} : S_{\triangle BOC} = 1:2\)

∵ \(S_{\triangle AOB} = 1\)

∴ \(S_{\triangle BOC} = 2\)

同理: \(S_{\triangle AOD}\) 与 \(S_{\triangle COD}\) (不同底, 同高)

且底之比为OA:OC=1:2

∴ \(S_{\triangle AOD} : S_{\triangle COD} = 1:2\)
