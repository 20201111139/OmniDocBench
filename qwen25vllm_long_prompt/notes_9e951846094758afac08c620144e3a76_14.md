$\text { 延伸拓展 } \quad \text { 求 } S_{\triangle A M D}=?$

$\therefore \frac{S_{\triangle A M D}}{S_{\triangle M B C}}=\left(\frac{\text { 相似比 }}{k}\right)^{2}=\left(\frac{4}{5}\right)^{2}=\frac{16}{25}$

即 $\frac{S_{\triangle A M D}}{S_{\triangle A M D}+S_{\text {四边形 } A B C D}}=\frac{16}{25} \quad \text { (1) }$

But. $S_{\text {四边形 } A B C D}=$ ?

有的同学可能会说四边形 $A B C D$ 为梯形

$=\frac{1}{2}(A D+B C) \times \text { 高 }=\frac{1}{2} \times(8+10) \times \text { 高 }$

$=9 \cdot \text { 高 } \quad \text { (1) }$

但是. 高是多少呢? 这就是问题所在!

看我的! 如图: 作 $D Q / / A B$. 交 $B C$ 于 $Q$

$\because A D / / B Q \quad A B / / D Q \therefore$ 四边形 $B Q D A$ 为平行四边形.

$\therefore B Q=A D=8 \therefore C Q=2 \quad D Q=B A=4$

在 $\triangle D Q C$ 中 $\left\{\begin{array}{l}Q D=4 \\ Q C=2 \\ C D=5\end{array}\right.$ 那面积呢?
