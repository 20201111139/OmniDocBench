# 行业经纬

## 凝冻来临,锦屏县沼气显“身手”

近期,贵州、江西、湖南等多个地区出现冰冻灾害,气温骤降,许多农村户用沼气池的安全使用面临挑战。贵州省锦屏县面对冰冻来袭,提前预防,及时维护,充分发挥沼气池的作用,给农民的生活、生产带来了极大的便利。

近日,贵州省锦屏县境内大范围地被冰雪覆盖,15个乡镇不同程度地遭遇凝冻灾害,其中气温最低的平秋镇达到零下5.2℃,农业、水利、电力等行业均不同程度受灾,大部分乡镇公路、桥涵的路面结冰,交通中断,水管、水表冻裂。锦屏县农村户用沼气池有14000多口,虽有一小部分被冰雪压坏了池子,或损坏了配件,但由于该县提前做了预防和准备,保障了大部分沼气池的安全使用,充分发挥了沼气对于农民生产生活的作用。

由于凝冻,锦屏县偶里乡云台村到311省道的输电线路被压断,村民们只能点煤油灯,老者在该村看到,因冰冻而停电的日子里,村民们的沼气灶具派上了大用场。吴才渊是当地一位小有名气的沼气池建设施工技术人员,据他介绍,自2008年的那场雪灾后,县里非常重视应对凝冻灾害的沼气技术,多次开展培训和宣传工作,他也因此成为“县解决雪灾天气沼气池安全使用技术指导员”。在凝冻来临前,当地在电视上通知村民们,天气可能突变,要及时做好沼气池的管理工作。于是他与全县90多名沼气池建设施工技术人员一起,走村串寨,对沼气池进行了全面检修。同时发放宣传资料,告诉村民们在冰冻天气下,做好沼气池的保暖防冻。如今,沼气应急照明,成了农民家中的“停电宝”。
