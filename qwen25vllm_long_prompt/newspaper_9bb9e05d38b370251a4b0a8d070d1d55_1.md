# 平江 农业综合开发助力“农民田间学校”

前不久，笔者来到湖南省平江县高标准农田建设示范项目区，看到的是一道道坚固的田埂，一条条排水沟纵横交错，一栋栋新建的农机库房，一处处美观实用的便民设施，一幅幅新农村的美景呈现在眼前。

我们来到“金星田间学校”培训现场，只见农民朋友分成几个小组，围坐在一起大声讨论水稻高产栽培技术、农业技术辅导员正给他们分发油菜苗标准，就提壮苗、育壮苗、高产的不同时期及影响因素进行指导。

2010年，该县实施国家农业综合开发高标准农田建设项目区共涉及面积1.1万亩，修建机耕路23公里，排灌站8座，开挖排灌渠21.8千米，修建系建筑物40个，机耕桥路4.8千米，绿化面积3400亩。该县在实施项目过程中把农田基础设施建设与提高农民综合素质相结合，在提高土地生产能力的同时，提高了农民的科技水平。在项目区内，他们与当地农业部门紧密配合，积极支持开办了四期农民田间学校，通过这一新的培训方式对项目的农民进行农业技术培训与推广，收到了良好的效果。

在培训过程中，以农民为学员，充分调研，了解掌握农民需要学什么，想问什么，做到自下而上。依托农技部门试验示范基地，以田间为课堂，以实践为手段，运用互动式、参与式方法组织培训，课堂分小组进行。要求参加培训的农民动口、动手、动脑，思考问题，解决问题。

新的培训理念、方法、方式，打破了传统的填鸭式、上课式、边听边记的培训模式，根据农民教育特点和学习规律，尊重农民，活跃气氛，交流经验，增强合作，加深理解，达到主动学习的目的与效果。

该县通过开办农民田间学校，项目区内，水稻主导品种、优质稻、超级杂交稻的应用率达到90%以上，测土配方施肥、水稻绿色防控综合技术、超级杂交稻高产栽培等主推技术得到了广泛推广，应用面积达到80%以上。

现在，项目区的农民李孝忠感慨地说：“农业综合开发修了路，建了机井，帮我们解决了农田生产的硬件设施。”农民田间学校的开办，新方式、新理念、传授农业新技术帮我们抓住了农业生产的“软件”建设。农业生产大有奔头了。
