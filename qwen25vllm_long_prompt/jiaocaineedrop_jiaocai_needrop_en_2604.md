3. 2 或 5 【解析】∵ $AB$ 是 $\odot O$ 的直径，∴ $\angle ACB = 90^\circ$。在 Rt $\triangle ABC$ 中，$AC = 4$，$BC = 3$，∴ $AB = \sqrt{4^2 + 3^2} = 5$。∵ $l \parallel AB$，∴ $\angle ACP = \angle CAB$。∵ 以点 $P, A, C$ 为顶点的三角形与 $\triangle ABC$ 相似，∴ $\frac{PC}{AB} = \frac{AC}{AC}$ 或 $\frac{AC}{AB} = \frac{PC}{AC}$，∴ $\frac{PC}{5} = 1$ 或 $\frac{4}{5} = \frac{PC}{4}$，解得 $PC = 5$ 或 3.2。综上可知，若 $\triangle ABC$ 与 $\triangle PAC$ 相似，则 $PC = 3.2$ 或 5。

4. 1 或 3 或 8 【解析】设 $AP = x$，则 $PB = 9 - x$。∵ 以 $A, C, P$ 为顶点的三角形与以 $B, D, P$ 为顶点的三角形相似，$\angle A = \angle B = 90^\circ$，∴ 分两种情况讨论：① 当 $\frac{AC}{BD} = \frac{AP}{PB}$ 时，$\frac{2}{4} = \frac{x}{9 - x}$，解得 $x = 3$。② 当 $\frac{AC}{BP} = \frac{AP}{BD}$ 时，$\frac{2}{9 - x} = \frac{x}{4}$，解得 $x = 1$ 或 8。∴ 当以 $A, C, P$ 为顶点的三角形与以 $B, D, P$ 为顶点的三角形相似时，$AP$ 的长为 1 或 3 或 8。故答案为 1 或 3 或 8。

5. $\frac{40}{9}$ 或 5 【解析】设 $BF = x$，∴ $BF = B'F = x$，∴ $FC = BC - BF = 10 - x$。∵ $\angle FCB' = \angle BCA$，∴ 当 $\frac{CF}{CB} = \frac{CB'}{CA}$ 时，$\triangle CFB' \sim \triangle CBA$，即 $\frac{10 - x}{10} = \frac{x}{8}$，解得 $x = \frac{40}{9}$；当 $\frac{CF}{CA} = \frac{CB'}{CB} = \frac{FB'}{AB}$ 时，$\triangle CFB' \sim \triangle CAB$，即 $\frac{10 - x}{8} = \frac{x}{8}$，解得 $x = 5$。综上所述，当 $BF = \frac{40}{9}$ 或 5 时，以点 $B', F, C$ 为顶点的三角形与 $\triangle ABC$ 相似。

6. ①④ 【解析】∵ 四边形 $ABCD$ 为正方形，∴ $\angle ADC = \angle BCD = 90^\circ$，$AD = CD$。∵ $E, F$ 分别为 $BC, CD$ 的中点，∴ $DF = EC = 2$，∴ $\triangle ADF \cong \triangle DCE$ (SAS)，∴ $\angle AFD = \angle DEC$，$\angle FAD = \angle EDC$。∵ $\angle EDC + \angle DEC = 90^\circ$，∴ $\angle EDC + \angle AFD = 90^\circ$，∴ $\angle DGF = 90^\circ$，即 $DE \perp AF$，故①正确。∵ $AD = 4$，$DF = \frac{1}{2}CD = 2$，∴ $AF = \sqrt{4^2 + 2^2} = 2\sqrt{5}$。∵ $\frac{1}{2}AD \cdot DF = \frac{1}{2}DG \cdot AF$，∴ $DG = \frac{AD \cdot DF}{AF} = \frac{4\sqrt{5}}{5}$，故②错误。∵ $H$ 为 $AF$ 的中点，∴ $HD = HF = \frac{1}{2}AF = \sqrt{5}$，∴ $\angle HDF = \angle HFD$。∵ $AB \parallel DC$，∴ $\angle HDF = \angle HFD = \angle BAG$。∵ $AG = \sqrt{AD^2 - DG^2} = \frac{8\sqrt{5}}{5}$，$AB = 4$，∴ $\frac{AB}{DH} = \frac{4\sqrt{5}}{5} = \frac{AG}{DF}$，∴ $\triangle ABG \sim \triangle DHF$，故④正确。∴ $\angle ABG = \angle DHF$，而 $AB \neq AG$，则 $\angle ABG$ 和 $\angle AGB$ 不相等，∴ $\angle AGB \neq \angle DHF$，∴ $HD$ 与 $BG$ 不平行，故③错误。故答案为 ①④。
