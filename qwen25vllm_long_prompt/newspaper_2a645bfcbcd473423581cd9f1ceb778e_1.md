# 全国农业工作会议召开

## 全国农业机械化工作会议提出——

### 十二五农机化水平超百分之六十

本报北京12月23日讯 记者王浩报道：12月23日，农业部在京召开全国农业机械化工作会议。会议的主题是，加快转变发展方式，强力推动农业机械化科学发展。会议回顾了“十一五”农业机械化发展成就，深入分析了“十二五”农业机械化发展形势，研究部署了“十二五”农业发展目标任务和主要措施，部署了2011年工作。农业部副部长张桃林出席会议并讲话。

会议指出，“十一五”期间，我国农业机械化发展取得了良好成绩，综合机械化水平由2005年的40%提高到2010年的52%，标志着我国农业生产进入机械化主导的新阶段。会议回顾了关于农业机械化发展的政策、工业支撑和快速发展的原因，总结了各地的成功经验和做法，部署了实现农机购置补贴155亿元资金的实施工作。

会议提出，2010年全国农业机械化水平比2005年提高了16个百分点。“十二五”是我国农业机械化发展环境显著变化、政策法规不断健全、发展程度明显加深、作用更加突出的关键时期，要牢牢把握发展的新机遇，紧紧围绕发展现代农业、建设社会主义新农村、夺取粮食和农业丰收、促进农民持续增收、保持农村社会和谐稳定的大局，着力优化农业机械化发展政策法规体系基本健全。二是农机购置补贴资金投入大幅增加，装备总量快速增长。三是农业机械化水平显著提高，农业生产经营方式实现了由人畜力作业为主向机械化作业为主的历史性跨越。四是农机作业服务蓬勃发展，农业社会化服务能力不断增强。五是农机理论体系初步建立，中国特色的农业机械化发展理论基本形成。

会议确定了“十二五”农机化发展目标和主要措施，提出到2015年，农业综合机械化水平达到60%以上，主要粮食作物机械化水平达到80%，装备结构明显改善，农机化作业水平大幅提高，农机化服务组织体系基本健全，农机化科技水平明显提高，农机化管理服务能力显著增强。会议还部署了2011年重点工作。
