# 重视农业就是重视民生的根本

## ——常州水稻单产连创江苏“八连冠”的启示

![Image]

#### 重视农业重视粮食重视投入稳夺“八连冠”

位于苏南经济板块的常州市，农业在GDP中所占份额不到4%。在这样一个典型的经济发达地区，农业份额虽小却举足轻重。市委书记范燕青说，无论经济、社会如何发展，农业永远都是无法离弃的基础产业，保证粮食安全不仅是经济问题，也是重大的政治问题。

近年来，常州市不断升级农田和粮食生产条件，持续加大各项强农惠农政策落实力度，除落实中央和省有关政策措施外，先后出台了水稻、良种、农机、农资增购补贴等政策，以及农业保险等一系列配套政策与扶持措施。2003～2010年投入2亿多元，用于高标准农田建设、水稻良种工程建设、水稻种植机械化补贴，组建专业化服务组织补贴、粮油高产增效创建等。其中，仅今年投入水稻稳产增效资金就超过5000万元，提高农业机械化作业水平，调动农民种粮积极性。

良种是农业生产的最重要基本条件，是粮食增产增收的原因。近年来，常州市加大优良品种引进、繁育、推广力度，水稻新品种如雨后春笋般涌现，高产新品种闻名全国的常州种子，来自常州的品种就占了4个。

“常州两季稻只要有一个广适性、高产高产品种，产量每年都有新突破，这是与政府对粮食生产的高度重视和有关政策分不开的。”江苏（武进）水稻研究所所长黄俊说，他的育种团队因品种供不应求价格提高而出现资金缺口，对此，市领导特事特办，在最短时间内解决问题，令他十分感动。目前，他所在任的江苏武进水稻研究所每年都能得到市县级财政200多万元的育种专项资金保证。

良种要有良法的土壤才能发挥增产性能。如何让有限的耕地创造出更多的“富”？常州市把改善粮食生产的“硬件”，为土地“增肥”作为提高粮食生产水平的根本性措施。常州市自2005年实施测土配方施肥以来，其核心就是提高农业综合生产能力与可持续发展能力。在这方面，政府舍得投入。2009年，常州市在全省率先提出建设现代农业“双百万亩”工程，提出用5年时间新建设一百万亩标准农田，目前全市标准农田比重已达到45%以上，亩均投入达到1850元，新建标准农田亩产10.97吨。

金坛市金城镇有31家砖瓦窑厂，这些“吃”地大户，如今却变成了380多亩高标准农田。该镇境内因烧窑成形的一条公里长的荒滩地，多年无人问津。2009年，金城镇将荒滩地作为高标准农田和高产建设，变成了优质农田，粮食亩产已达到500公斤以上。2010年，金坛市水稻种植面积30.92万亩，亩产622公斤，高标准农田建设在稳增产上发挥了不可替代的作用。

# 兽医工作再上新台阶

今年以来，面对国内外动物疫情形势复杂多变、自然灾害多发频发等一系列严峻挑战，各级畜牧兽医部门认真贯彻落实中央和农业部党组部署，确保全年没有发生区域性重大动物疫情，有效保障了畜牧业生产安全、动物产品质量安全和公共卫生安全，为落实动物疫病防控措施，努力确保不发生区域性重大动物疫情，保障防疫工作，落实免疫监测、流调、消毒等综合防控措施，禽流感、高致病性蓝耳病、口蹄疫等猪瘟4种重大动物疫病免疫密度均达到95%以上，抗原合格率达到国家规定要求。一些省份发生自然灾害发生后，各疫病部门迅速启动应急机制，开展疫情排查，落实各项措施，确保死亡动物无害化处理、消毒和紧急免疫工作，确保了“大灾之后无大疫”。

# 畜牧业发展亮点多成效大

2018年畜牧业发展亮点多、成效大。生猪生产稳步发展，奶业恢复性增长，饲料工业持续向好，兽医工作再上新台阶，应对灾害有力，应对突发事件有力有效，畜产品质量安全水平不断提高，为保持农业农村经济发展良好势头和国家宏观调控预期作出了重要贡献。

围绕完成了保障畜产品有效供给的目标任务，实现了产业平稳发展。今年，各地开展标准化养殖示范创建活动，千方百计基础母猪，规模养殖，调结构生产，努力保持畜牧业平稳发展。生猪生产稳步回升，奶业恢复性增长，饲料工业持续向好，兽医工作再上新台阶，强化监测预警，实施疫情排查，落实各项措施，确保死亡动物无害化处理、消毒和紧急免疫工作，确保了“大灾之后无大疫”。

# 柑橘院士“秀”柑橘

本报记者 何卫卫 孙维雄近日，在位于湖北省武汉市的华中农业大学国家柑橘育种中心的室外育种圃里，举行了一场别开生面的“橘展”。在“柑橘院士”、中国工程院院士、华中农业大学校长邓秀新教授带领下，前来参观的嘉宾们，为众人讲解。

在摆满35个不同品种柑橘果实的华中农业大学橘园会议室，邓秀新又说起我国柑橘的科技成就。他说，这个“技术体系”在柑橘产业“基础性工作”、“重点工作”、“潜性工作”和“应急性工作”中发挥了巨大功能，有效促进了科技与生产相结合，有效推动了柑橘产业的优化、化、引导和无害化栽培等先进生产技术在各个产区起到了柑橘品种的良化。此外还相继研究出高接换种、地覆膜等10多项配套技术并广泛推广，仅湖北省秭归县就有6万多亩老橘园完成高接换种，实现了4年投产3年丰产。

# 吴大爷的四个“小本本”

说起近年来的农村新变化，山东省东营市垦利区农安镇今年78岁的吴金奎老人拿出了4个“小本本”。尽管吴大爷识字不多，但对每个“小本本”的用途却是一清二楚。他指着第一个“小本本”说：“这是养老金本，每月能领到360元，另外俺们村里还有70岁以上老人每月能领到300元，俺两口子每月光养老金就能领到1200元。”

吴大爷的第二个“小本本”是“新农合”本。“有了这个本，看病不用愁，住院报销比例还挺高。”吴大爷说。2009年，吴大爷两口子看病没花多少钱，可是报销后，负担明显减轻了。

第三个“小本本”便是银行存折了。吴大爷的3个儿女孝顺都没想到，仅三年时间，他们脚下这片土地因为不再农药、化肥施用，种出来的农产品价格翻了倍，甚至十几倍。
