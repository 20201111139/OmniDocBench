# 法治时空

## 公安部开展“亮剑”专项行动

### 严打8类侵犯知识产权犯罪

记者从公安部日前举行的新闻发布会上获悉，自11月19日公安部部署开展“亮剑”专项行动以来，截至11月30日，全国公安机关已破获侵犯知识产权和制售伪劣商品犯罪案件676起，案值8.83亿元，抓获犯罪嫌疑人1586名。

据公安部“亮剑”专项行动领导小组办公室主任、经济犯罪侦查局局长孟庆丰介绍，此次专项行动的打击重点是：侵犯商标权、版权、专利权和植物新品种权等侵犯知识产权犯罪和生产、销售伪劣商品犯罪。其中，突出严厉打击以下8种犯罪活动：印刷复制假冒注册商标、光盘，非法出版、印刷、复制、销售侵权盗版图书、音像制品、电子出版物、软件、游戏产品等侵犯著作权的犯罪；跨地区（跨境）的侵犯知识产权、制假售假犯罪；规模化、组织化的侵犯知识产权、制假售假犯罪；与商业贿赂等其他犯罪相关的侵犯知识产权犯罪；其他情节严重、影响恶劣的犯罪。

孟庆丰表示，针对当前侵犯知识产权和制售伪劣商品犯罪专业化、规模化、跨区域化、网络化等特点，在“亮剑”行动中，公安机关将加强与有关部门的协作配合，形成打击合力，向上深挖生产源头，向下查清销售网络，集中捣毁了一批制假售假犯罪窝点，现场缴获了一批制假售假犯罪工具和犯罪团伙，斩断了一条制假、生产、储运、销售、出口的犯罪产业链条。截至目前，公安机关共捣毁制假售假窝点342个，缴获假冒注册商标标识171.17万件（套），假冒伪劣商品14.4万件，涉案金额达8.83亿元。

孟庆丰指出，侵犯知识产权犯罪不仅严重扰乱市场经济秩序，而且严重损害权利人的合法权益，破坏我国的国际形象。因此，公安机关将始终保持对侵犯知识产权犯罪的高压态势，依法严厉打击各类侵犯知识产权犯罪活动，切实维护人民群众的合法权益，为经济社会发展创造良好的法治环境。

### 法援为外来讨薪农民工撑腰
