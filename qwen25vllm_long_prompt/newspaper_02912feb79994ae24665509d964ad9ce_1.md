# 北大荒 中国农业机械化的领跑者

## 三台“火犁”起家 向着农机现代化疾跑

1947年，解放战争的炮火还在几百公里外隆隆作响，6月的一天，一辆从未见过的汽车驶来李大人、刘罗等4名军人，以略带神秘的口吻说，此刻，他们将执行一个特殊任务——受党中央、毛主席要求，在东北根据地组建一批粮食加工厂，总结经验，培训干部。东北农民，为将来实现农业机械化做准备。两人被任命为中国第一座“粮食工厂”——黑龙江省合江省第一农场（平安农场）场长和副场长。他们调集的全部家当只有2台烧油的拖拉机和3台旧日制开沟机扔下的“火犁”。

6月13日，刘罗驾驶隆隆开起“火犁”，在千亩荒原上拉出了“东方第一犁”，共和国的农业事业和农业机械化的篇章就此翻开。

此后，通北农场、友谊农场、八五六农场、八五八农场、八五九农场相继创建，标志着日本开拓团走下和从“白旗”接收的大批整套机械被收编，开始了艰苦的创业历程。

1952年12月，苏联援建的163个项目之一，以机械化为主要手段的大型农场开始建设，友谊农场的面积达20万公顷，是世界上最大的农场。1954年，友谊农场的“三夏”作业，当时国家来参观友谊农场的提出的任务是“出白粮”，出好粮，出人才”。

这一时期，中国的农业化进程快速进展。1959年，中国仿造出第一台—54履带拖拉机在北大荒投入使用。1961年，国产“东方红”系列拖拉机开始大批量生产，迅速推广到北大荒各大农场。中国终于开始了规模化的农业机械化，北大荒也成为中国农业机械化最早的实验基地。

时间的指针转到1977年夏天，“文革”后重建的黑龙江省国营农场总局召开“农业机械化会议”上看到国外各种先进的农机具时，大为震惊，“过去总觉得地区已经实现了机械化，现在一比，我们的农机设备比人家落后了将近半个世纪，我们再也不能自欺欺人，闭关自守了。”

“大马力”不断刷新产粮纪录

中国农业现代化看中国农场，中国农场看黑龙江。在黑龙江，可以看到世界上最先进的各种农业机械。迪尔、凯斯、雷诺、久保田、洋马……黑龙江垦区不断刷新着农机纪录，始终追踪世界最先进的技术。可以说，这里的农业现代化水平不仅领先全国，在世界范围内也处于先进行列。据一位“了解农业情况的人”说，中国农业机械化水平和世界上比起来总是差那么一点，即使在最好的农业机械上，它的技术也不见得比世界最先进的农场差，但产量上，375马力甚至550马力的大马拉拖拉机比比皆是。

“十五”期间，黑龙江垦区农机更新的总投资达54.5亿元，年均“十五”期间2.8倍，新增、改造内外作业机械24.3万台套，其中，建设现代农业装备示范农场33个，新装备、改造和经济作物生产机械化也取得了显著进展。高度的农业机械化，提高了农业抗灾能力，有效地提高了土地产出率、资源利用率和劳动生产率。
