# 科教周刊

## 农业部科技教育司指导

### 农业信息智能服务技术助力现代农业
——访中国农科院农业信息研究所所长许世卫研究员

#### 本报记者 周锋

现代化农业离不开信息、发达的国家经验表明，农业信息化是现代农业的重要标志。当前，我国已进入加快传统农业改造，走中国特色农业现代化道路的关键时期，充分发挥农业信息智能服务技术的支撑和替代作用，是我国农业信息领域专家、中国农业科学院农业信息研究所所长许世卫提出的观点。记者日前就有关问题，采访了许世卫研究员。

**记者：信息化已成为我们这个时代新的标志，成为推动经济社会发展的重要力量。有专家提出信息化与农业现代化的融合是大势所趋，您对此有何看法？**

**许世卫：** 当前，随着我国全面小康社会建设的大力推进，加快农业发展方式转变，大力推进农业信息化建设已经成为当务之急。农业信息智能服务技术是现代农业信息化的核心内容。农业信息智能服务技术是通过信息技术对农业信息进行采集、分析、加工、传播和使用的全过程，是表征农业“信息化”的过程，是把“信息”变成农业生产力的过程。通过农业信息智能服务技术在农业发展中的应用，可以有效解决农业信息不对称问题，提高农业生产的效率和效益。

**记者：我国正面临一个新的农业发展节点，机遇与挑战并存。在发展现代农业过程中，遇到了农业劳动力不断流失、农业生产成本持续上升、农产品价格波动等问题。农业信息智能服务技术能带来哪些变化，走好这条路？**

**许世卫：** 通过农业信息智能服务技术，可以优化农业资源配置，降低劳动成本上升问题；可以通过现代信息技术手段，对农业生产过程进行风险预警，早期预报，早期干预，防止自然灾害的突发性、大范围、大损失，保障农业生产的稳定性和持续性。同时，通过农业信息智能服务技术，可以有针对性地对农业信息进行采集、分析、加工、传播和使用，从而有效解决农业信息不对称问题。
