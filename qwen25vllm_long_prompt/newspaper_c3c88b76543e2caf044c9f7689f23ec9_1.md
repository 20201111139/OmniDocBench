# 农民利益的守护神

## 记湖南省农民负担监督管理领导小组办公室主任何仁芽

### 本报记者 吴砾星 文/图

#### 何仁芽的“三不”原则

何仁芽在工作中始终坚持“三不”原则：不接受宴请、不收礼、不报销任何费用。他经常深入基层调研，了解农民负担情况，为农民排忧解难。有一次，他在下乡调研时，发现某县存在严重的农民负担问题，立即向当地政府提出整改意见，并督促其落实。他还多次到贫困地区慰问困难群众，送去慰问金和生活用品。
