# Functional Groups in Organic Chemistry

Functional groups are the characteristic groups in organic molecules that give them their reactivity. In the formulae below, R represents the rest of the molecule and X represents any halogen atom.

| Functional Group | Formula | Example |
| --- | --- | --- |
| Hydrocarbons |  |  |
| Alkane | \( \text{R}^1-\text{C}-\text{R}^2 \) | Naming: -ane e.g. ethane |
| Alkene | \( \text{R}^1=\text{C}=\text{R}^2 \) | Naming: -ene e.g. ethene |
| Alkyne | \( \text{R}^1\equiv\text{C}\equiv\text{R}^2 \) | Naming: -yne e.g. ethyne |
| Arene | \( \text{R}^1-\text{C}=\text{C}-\text{R}^2 \) | Naming: -yl benzene e.g. ethyl benzene |
| Haloalkane | \( \text{R}-\text{X} \) | Naming: halo e.g. chloroethane |
| Alcohol | \( \text{R}-\text{OH} \) | Naming: -ol e.g. ethanol |
| Aldehyde | \( \text{R}-\text{C}=\text{O} \) | Naming: -al e.g. ethanal |
| Ketone | \( \text{R}^1-\text{C}=\text{O}-\text{R}^2 \) | Naming: -one e.g. propanone |
| Carboxylic Acid | \( \text{R}-\text{COOH} \) | Naming: -oic acid e.g. ethanoic acid |
| Acid Anhydride | \( \text{R}-\text{C}=\text{O}-\text{O}-\text{C}=\text{O}-\text{R} \) | Naming: -oic anhydride e.g. ethanoic anhydride |
| Acyl Halide | \( \text{R}-\text{C}=\text{O}-\text{X} \) | Naming: -oyl halide e.g. ethanoyl chloride |
| Ester | \( \text{R}-\text{C}=\text{O}-\text{O}-\text{R}^2 \) | Naming: -yl oate e.g. ethyl ethanoate |
| Ether | \( \text{R}-\text{O}-\text{R}^2 \) | Naming: -oxy -ane e.g. methoxyethane |
| Epoxide | \( \text{R}^1-\text{C}=\text{C}-\text{O}-\text{R}^2 \) | Naming: -ene oxide e.g. ethylene oxide |
| Amine | \( \text{R}-\text{NH}_2 \) | Naming: -amine e.g. ethanamine |
| Amide | \( \text{R}-\text{C}=\text{O}-\text{NH}_2 \) | Naming: -amide e.g. ethanamide |
| Nitrate | \( \text{R}-\text{O}-\text{N}=\text{O} \) | Naming: -yl nitrate e.g. ethyl nitrate |
| Nitrite | \( \text{R}-\text{O}-\text{N}=\text{O} \) | Naming: -yl nitrite e.g. ethyl nitrite |
| Nitrile | \( \text{R}-\text{C}=\text{N} \) | Naming: -nitrile e.g. ethanenitrile |
| Nitro | \( \text{R}-\text{NO}_2 \) | Naming: -nitro e.g. nitromethane |
| Nitroso | \( \text{R}-\text{N}=\text{O} \) | Naming: -nitroso e.g. nitrosoethane |
| Imin | \( \text{R}-\text{N}=\text{C}=\text{O} \) | Naming: -imine e.g. ethanimine |
| Imide | \( \text{R}-\text{C}=\text{N}-\text{C}=\text{O} \) | Naming: -imide e.g. succinimide |
| Azide | \( \text{R}-\text{N}=\text{N}=\text{N} \) | Naming: -yl azide e.g. phenyl azide |
| Cyanate | \( \text{R}-\text{O}-\text{C}=\text{N} \) | Naming: -yl cyanate e.g. methyl cyanate |
| Isocyanate | \( \text{R}-\text{N}=\text{C}=\text{O} \) | Naming: -yl isocyanate e.g. methyl isocyanate |
| Azo Compound | \( \text{R}-\text{N}=\text{N}-\text{R}^2 \) | Naming: azo- e.g. azoethane |
| Thiol | \( \text{R}-\text{SH} \) | Naming: -thiol e.g. methanethiol |
| Sulfide | \( \text{R}-\text{S}-\text{R}^2 \) | Naming: sulfide e.g. dimethyl sulfide |
| Disulfide | \( \text{R}-\text{S}-\text{S}-\text{R}^2 \) | Naming: disulfide e.g. dimethyl disulfide |
| Sulfoxide | \( \text{R}-\text{S}=\text{O} \) | Naming: sulfoxide e.g. dimethyl sulfoxide |
| Sulfone | \( \text{R}-\text{S}=\text{O} \) | Naming: sulfone e.g. dimethyl sulfone |
| Sulfonic Acid | \( \text{R}-\text{SO}_3\text{H} \) | Naming: -sulfonic acid e.g. benzenesulfonic acid |
| Sulfonate Ester | \( \text{R}-\text{SO}_3\text{O}-\text{R}^2 \) | Naming: -sulfonate e.g. methyl benzenesulfonate |
| Thiocyanate | \( \text{R}-\text{S}-\text{C}=\text{N} \) | Naming: -yl thiocyanate e.g. ethyl thiocyanate |
| Isothiocyanate | \( \text{R}-\text{N}=\text{C}=\text{S} \) | Naming: isothiocyanate e.g. ethyl isothiocyanate |
| Thial | \( \text{R}-\text{S}-\text{C}=\text{S} \) | Naming: -thial e.g. ethanethial |
| Thioketone | \( \text{R}-\text{C}=\text{S} \) | Naming: -thione e.g. propanethione |
| Phosphine | \( \text{R}-\text{P}=\text{H} \) | Naming: phosphine e.g. methylphosphine |
