(2) “・i” 的几何意义: 逆时针旋转 \(90^{\circ}\) 。
例如, \(3+2 \mathrm{i} \stackrel{\text { 逆时针旋转 } 90^{\circ}}{=} \quad(3+2 \mathrm{i}) \cdot\) \(\mathrm{i}=-2+3 \mathrm{i}\) 。

\[
Z(3,2) \stackrel{\text { 逆时针旋转 } 90^{\circ}}{\longrightarrow} Z^{\prime}(-2,3)
\]
