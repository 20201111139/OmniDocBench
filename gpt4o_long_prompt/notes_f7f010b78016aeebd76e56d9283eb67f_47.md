11) so

他说什么, 我也做什么, 作与3句互换表达语。

- He: I don't think so. (对此我不发表意见。)

- He lost a book. So did I. (他失去了一本书。我也是。)

12) a lot of / lots of / a number of / large numbers of / a great deal of / plenty of

"a lot of 无变化功能字与名词的拼"与" A large number of 不加s就表起到的有数与不少有个的意义,"学。

A lot of / lots of 的应用与复数名词及不可数名词相同的可以转换为副词表示数量与多或参与数量的意义, 与 much, so many, plenty of 同等, & many of / a number of / large number of, 动名词形式名词及贺量词和动词使用释用意义, 副词在 some, many, a lot of, plenty of a great deal of, plenty of 不变化功能基本的用途, 但可指定对相应副词作必要的应用, 而副词在副词进行数量方位致关系中也固定有分。

- A: A lot of people think that time is money. (许多人认为时间就是金钱。)  

- I don't have to do in a hurry because I have plenty of time.  
  (我不必匆忙去做，因为我有的是时间。)

- I have a number of letters to write today.  
  (今天有若干信要写。)

- I spend a great deal of time / money on shopping.  
  (我花费大量时间/金钱在购物上。)
