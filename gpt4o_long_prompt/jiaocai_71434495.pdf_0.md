1. 求定积分 \(\int_0^{\frac{\pi}{2}} \frac{\sin x \cos^2 x}{3 + \cos^2 x} \, dx\).

   **[答案]** 令 \( t = \sin^2 x \), 则 \( dt = 2 \sin x \cos x \, dx \), 于是
   \[
   I = \frac{1}{2} \int_0^1 \frac{t \, dt}{2 + t} = \frac{1}{2} \left[ \ln (2 + t) \right]_0^1 = \ln 2 - \frac{1}{2}
   \]

2. 计算二重积分 \(\iint_D \sin (x^2 + y^2) \, dx \, dy\), 其中区域 \(D = \{(x, y) \mid 0 \leq x, y \leq \sqrt{\pi}\}\).

   **[解答]** 解析区域 \(D_1 = \{(x, y) \mid 0 \leq y \leq x \leq \sqrt{\pi}\}\), 由极坐标可知
   \[
   \text{原式} = 2 \int_0^{\sqrt{\pi}} \sin x^2 \, dx = 2 \int_0^{\sqrt{\pi}} x \sin x^2 \, dx = \int_0^{\pi} \sin x^2 \, dx = 2
   \]

3. 求级数 \(\sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n(2n+1)}\) 的和。

   **[答案]** 解析步骤
   \[
   \sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n(2n+1)} = \sum_{n=1}^{\infty} \left( \frac{1}{2n} - \frac{1}{2n+1} \right) = \ln 2 + \frac{\pi}{4} - 2
   \]

   **[答案]** 证明 (1) 注意到 \( f(0, 0) = 0\), \(\frac{\partial f}{\partial y}(0, 0) = -1\), \(\frac{\partial^2 f}{\partial y^2}(0, 0) = 6 + 12xy - 3xy^2 - 6x^2 + 6xy - 12xy^2 = 0,\)

   \( \frac{\partial^2 f}{\partial x \partial y}(0, 0) = -6 + 6xy + 12xy^2 + 3xy^2 - 6 = 0\)

   \( \frac{\partial}{\partial x}\left(\frac{\partial f}{\partial y}\right)(0, 0) = -6y - 12xy^2 - 6xy^2 - 6xy^2 = 0\)

   \( \frac{\partial}{\partial y}\left(\frac{\partial f}{\partial x}\right)(0, 0) = 3xy + 6xy^2 = 0\)
