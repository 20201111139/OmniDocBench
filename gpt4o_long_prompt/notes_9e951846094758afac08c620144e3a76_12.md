1. DE // BC   
   - \( AN = AP = DN \)  
   - \(\triangle ADN \sim \triangle ABM\)   
   - \(\triangle AEN \sim \triangle AMC\)  

   \[
   \frac{AN}{AM} = \frac{AP}{AB} = \frac{BN}{BM}

2. \( AN = 2NM \)   
   \( \frac{AN}{AM} = \frac{2}{3} \)  
   - 故两组相似关系相似

3. \[ \frac{S_{ADM}}{S_{ABM}} = \left(\frac{2}{3}\right)^2 = \frac{4}{9} \]  
   \[ \frac{S_{AMC}}{S_{ABC}} = \left(\frac{3}{2}\right)^2 = \frac{9}{4} \]

   - 改 \[ S_{ADM} = 4k \]  
   - \[ S_{AME} = 4t \]  
   - \[ S_{ABM} = 9k \]  
   - \[ S_{AMC} = 9t \]  
   - \[ S_{ABC} = S_{ABM} + S_{AMC} = 9k + 9t \]

4. \( S_{ADM} = \frac{1}{2} S_{ABM} \)  
   - 角不等夹  
   \( S_{AME} = \frac{1}{2} S_{AMC} \)  
   - 高相等  

5. \( S_{ADM} = 2k \)  
   - \( S_{AME} = 2t \)  
   - \( S_{ADE} = S_{ADM} + S_{AME} = 2k + 2t \)  
   - \( S_{ABC} = 9k + 9t = 9(k + t) \)
