# 1. 运用各种描写手法来塑造人物形象。

<table>
    <tr>
        <td>描写手法</td>
        <td>类别</td>
        <td>具体分析</td>
    </tr>
    <tr>
        <td rowspan="2">正面描写</td>
        <td>肖像描写</td>
        <td>特征、作用</td>
    </tr>
    <tr>
        <td>肖像描写是对人物形象的外部特征的描写，包括人物的容貌、身材、服饰等。它对于人物性格和人物形象的完整体现，有着重要的表现作用。人 物肖像描写入手，可以迅速掌握人物的外在特征、身份、地位、教养，甚至内在性格。小说人物的肖像描写可以是相对静止的整体提示，如可比喻的个人物特写镜头，如正面刻画人物肖像，如《林黛玉进贾府》中林黛玉的肖像描写；也可以是对某一局部的重点刻画，如鲁迅在《祝福》里对祥林嫂的眼神的三次描写。</td>
    </tr>
    <tr>
        <td>正面描写</td>
        <td>语言描写</td>
        <td>小说的语言描写包括对话和独白。个性化的对话，可以显示人物的不同性格。鲁迅笔下人物的语言，都极有性格特征，孔乙己的渲染斯文，一喝痴的夹刻利害，阿 Q 的自欺自足，无不各形各色，跃然纸上。如有特点的内心独白，也是透视人物内心秘密、展现人物性格特征的一种重要手段。如《守财奴》中葛朗台在得知妻子病情加重，财产有可能失去时，情不自禁地对白己说道：“那商量是朱白的脖子!”它直接揭示了人物对金钱赤裸裸的占有欲。</td>
    </tr>
    <tr>
        <td rowspan="2">正面描写</td>
        <td>动作描写</td>
        <td>特征、作用</td>
    </tr>
    <tr>
        <td>动作描写是指作家对人物富有性格特征的动作所做的生动、具体的描绘和刻画。人的行为动作是人物的思想感情的外在显现，是人物性格的具体展现。如《林黛玉进贾府》中“便不走了，只向丫鬟边椅子上坐了”就表现了林黛玉的小心谨慎。</td>
    </tr>
    <tr>
        <td>正面描写</td>
        <td>心理描写</td>
        <td>心理描写直接表现人物思思想和内在情感（矛盾、焦虑、担心、喜悦、兴奋等），表现人物思想品质，刻画人物性格，推动情节发展，可以使人物心理更加鲜明、生动、具体。在人物描写具有的重要意义。视《老人与海》中对桑提亚哥的心理描写：他在大海中与鲨鱼搏斗，胜利的信念驱使他无畏和勇敢：“没受过一步历考好，但我没有得到这条鱼。”而而他又坚信：“人可以被毁灭，但不能给予败。”这些心里描写生动地表现出桑地亚哥敢于放弃冷机会，绝不屈向命运低头，即使失败了，同样也坦荡荡，不失-重臣下的优雅风度。</td>
    </tr>
    <tr>
        <td colspan="2">侧面描写</td>
        <td>细节描写是指对文学作品中的人物、不琐要求于所做的具体、深入、某一细微事物的刻画，能更细腻地展示人物的特征。如《林黛玉进贾府》中林黛玉对“读书”一问题的微次不同的回答，就展现了林黛玉的小心谨慎的性格。</td>
    </tr>
    <tr>
        <td colspan="2">侧面描写</td>
        <td>侧面描写又叫间接描写，是从侧面烘托人物形象，通过对其周围人物或环境的描绘来表现所要描写的对象，以使其鲜明突出。</td>
    </tr>
</table>
