## 股票投资评级说明

### 评级

| 评级          | 说明                       |
| ------------- | -------------------------- |
| 买入（Buy）   | 预计相对强于市场表现20%以上      |
| 增持（outperform） | 预计相对强于市场表现5%~20%     |
| 中性（Neutral）     | 预计相对市场表现是-5%~+5%之间波动 |
| 减持（underperform）| 预计相对弱于市场表现5%以下     |

### 行业评级

| 评级             | 说明                 |
| ---------------- | -------------------- |
| 看好（overweight） | 预计行业趋势整体于市场表现  |
| 中性（Neutral）     | 预计行业趋势整体于市场表现基本持平 |
| 看淡（underperform）| 预计行业弱于整体市场表现    |

备注：评级系指为以提示占后的6~12个月内，证券相对市场态度潜在的涨跌幅表现。其中A股斑益指数为沪深300指数，港股主要指数为报指基线，新能源主要指数为三成指数（针对沪版指数活动）、美三板市场指数（针对带市报告的）。美减基指数为基线构索 500 成分频制造综合指数。我们在此提发您，不可谓系所承托所承不同的评级体系及评级报者。我们采用的承相评份依承系，表示读家的一种比理论说明。投资者人人及者家由证监的决定取决于个人的实际情况，比中心当前特约特色抽以及其他您录考考虑的因素。投资者应间供整在属核，以获取比短股票市场观点与信念，反应仅仅在评级变化建议。
