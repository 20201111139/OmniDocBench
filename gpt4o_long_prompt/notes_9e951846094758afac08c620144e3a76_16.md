7. 如图，梯形 \(ABCD\), \(AB \parallel CD\). 若 \(AB = 1\), \(CD = 2\), \(AC = 6\), \(S_{\triangle AOB} = 1\), 求 \(AD = ?\).

求 \(S_{梯形 ABCD}\)：

解：

∵ \(AB \parallel CD\)

∴ \(\triangle AOB \sim \triangle COD\)

即 \(\frac{AO}{CO} = \frac{AB}{CD}\)

∴ \(AB = 1, \, CD = 2, \, AC = 6\)

\[ \frac{AO}{CO} = \frac{1}{2} \Rightarrow CO = 2 \cdot AO \]

又：

\[
AC = AO + CO = 3 \, AO \Rightarrow 3 \, AO = 6 \Rightarrow AO = 2
\]

又：

\[
\frac{S_{\triangle AOB}}{S_{\triangle COD}} = \left(\frac{AB}{CD}\right)^2 = \left(\frac{1}{2}\right)^2 = \frac{1}{4} = \frac{1}{S_{\triangle COD}}
\]

\[ S_{\triangle COD} = 4 \]

∴ \(S_{\triangle AOB} \) 与 \( S_{\triangle BOC} \)（同底、同高）

即使 \(BO \, CO = 1:2\)

\[ S_{\triangle AOB} : S_{\triangle BOC} = 1:2 \Rightarrow S_{\triangle AOB} = 1 \]

即：

\[ S_{\triangle BOC} = 2 \]

同理：

\[ S_{\triangle AOD} \sim S_{\triangle COD} \, (同底、同高)\]

即使得出 \(OA : OC = 2:1\)

\[ S_{\triangle AOD} : S_{\triangle COD} = 1:2 \Rightarrow S_{\triangle COD} = 1 \]

即得出：

\[ OA : OC = 2:1 \Rightarrow S_{\triangle AOD} : S_{\triangle COD} = 1:2 \]
