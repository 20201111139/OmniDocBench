# 1.2 中国领先生物制药和生物科技公司迎来了资金密集投资者

尽管整体交易市场不畅，但跨境交易的数量在2019年相比仍有所增加，甚至超过了2018年的水平，其中主要是入境交易的增加（图4和图5）。

中国曾在批准新冠疫苗申请快速拨号的数国家之一，即领先生物制药和生物制药公司吸引了全球资本。生物制药和生物科技行业交易因时事变更剧烈。2020年入境交易及全球私募权及风投数机构的资金，平均交换金额超过15亿美元。全球技术的流入影响了国际私募权权机构对中国生物制药和生物科技行业的信心。藉疫苗声明，两大投资目标成为稳健生物医药投资保留点和非显障性之一由2021年代便可达成实力力量。微创医疗新的明佳证券（上海）成立扩在首达证券交易所的首次公开募股，而朗生药品和北海药业继续完成其在香港交易所的首次公开募股申请。
