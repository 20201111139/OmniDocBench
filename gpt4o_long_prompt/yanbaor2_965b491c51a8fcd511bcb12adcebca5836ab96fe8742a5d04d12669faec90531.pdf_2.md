# 全球房地产市场的并购趋势

## 办公

宏观经济环境对办公资产的影响仍在持续，全球空置率达到两位数。利率上调抬高了收购和建设的债务成本，并对潜在回报率造成下行压力。随着建筑物等级成为获得资本和提升资产财务表现愈来愈重要的决定因素，短期内融资将面临更大挑战。

我们预计，在包括部分亚洲重点城市、地区优越以及现代化和高质量建筑群所在的地区，办公资产需求将保持旺盛，并购交易也将最为活跃。尽管到目前为止，大多数办公员工回归办公室工作的速度慢于预期，但业主们依然对利润增长的回归趋势抱有信心。

在欧盟，若现有规章规定必须达到一定标准，这一法规调整刺激了市场对优质资产的需求。如2023年（英国）和2030年（美国）起，办公楼必须拥有能源绩效证书（EPC）C级或以上。随着越来越多的公司致力于实现碳中和，提出明确的净零目标，出租方也做出相应调整，因此，对予以采用绿色科技、配备专门设施以改善租户体验的新型办公空间的需求随之上升。
