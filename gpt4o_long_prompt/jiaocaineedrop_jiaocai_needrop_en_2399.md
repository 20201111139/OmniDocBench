# 全书大归纳

## 核心知识归纳

### 一元一次方程

<table>
  <tr>
    <td>知识点</td>
    <td>内容</td>
  </tr>
  <tr>
    <td>方程的解的概念</td>
    <td>
      使方程左、右两边的值相等的未知数的值就是该方程的解。<br>
      提示：将方程的解代入方程，等式成立。
    </td>
  </tr>
  <tr>
    <td>等式的本性</td>
    <td>
      (1) 等式两边都加上 (或都减去) 同一个数或同一个整式，所得结果仍是等式。
      如果 \(a = b\)，那么 \(a + c = b + c, a - c = b - c\)；<br>
      (2) 等式两边都乘以 (或都除以) 同一个数 (除数不能为 0)，所得结果仍是等式。
      如果 \(a = b\)，那么 \(ac = bc, \frac{a}{c} = \frac{b}{c} (c \neq 0)\)
    </td>
  </tr>
  <tr>
    <td>方程的变形规则</td>
    <td>
      (1) 方程两边都加上 (或减去) 同一个数或同一个整式，方程的解不变；<br>
      (2) 方程两边都乘 (或都除以) 同一个不等于 0 的数，方程的解不变
    </td>
  </tr>
  <tr>
    <td>移项</td>
    <td>
      将方程中的某项改变符号后，从方程的一边移到另一边的变形。<br>
      注意：移项必须改变符号
    </td>
  </tr>
  <tr>
    <td>一元一次方程的定义</td>
    <td>
      只有含有一个未知数，并且含有未知数的式子都是整式，未知数的次数都是 1，
      像这样的方程叫做一元一次方程。<br>
      提示：一般又称为 \(ax = b \ (a, b\) 为常数, \(a \neq 0\))
    </td>
  </tr>
  <tr>
    <td>解一元一次方程步骤</td>
    <td>
      (1) 去分母；(2) 去括号；(3) 移项；(4) 合并同类项；(5) 将未知数的系数化为 1。<br>
      提示：应用移方程特点，灵活选择解题步骤
    </td>
  </tr>
</table>
