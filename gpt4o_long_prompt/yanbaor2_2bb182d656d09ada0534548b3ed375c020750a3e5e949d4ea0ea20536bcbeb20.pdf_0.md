# 中国创新崛起  
——创新生态孕育创新生机  

文 / 刘明华 冯晦 李奕虹  

在经济下行的大环境下，为了保持自身优势，世界各国纷纷加大国家推动科技创新的力度，创新已成为各界关注的焦点。中国政府的高度重视创新也为建设创新型国家奠定了重要的基础。中国的创新正逐步进入“深水区”。中国的创新生态到底是什么？

创新生态发展在面临各种机遇和挑战的同时，也对政府及企业提出了更多的需求。显然，中国急需的是一问迅速发展。  

中国的创新正在进入新的发展阶段。任何产业的创新发展需要三个要素——一是集聚的大科技企业和共享的国宝资源；二是规模比较快的大市场，三是创新所需要的巨额技术投入。未来，中国将会迎来更多创新的产品和服务面世界。
