# 重点提示  

$\diamond$  荐菜形态似微型睡 莲，花虽小，但密集成片，有着不同情调的景观效果。在长江流域及以南地区的冬季，菜还常以常绿 或半常绿形式生长；在炎热的夏季长势缓慢，在浅水水域甚至出现枯黄现象。  

$\diamond$ 纯菜属于一种高贵优良的水生蔬菜，但产出比不高。在工程应用上，可结合其经济功能综合考虑配置。  

# 品种、类似种  

荐菜属有数个原生品种，外形差异不大。有景观效果类似的同属植物一金银莲花（N.indica）、水皮莲（N.cristatum）等，花为白色，叶形与荐菜差异不大。有景观效果近似的不同科属品种一菜（Braseniaschreberi），叶为椭圆形，与荐菜叶形有区别，花较小，红褐色。  

# 工程应用  

菜叶片翠绿，小黄花密集精巧别致，是优良的小型浮叶景观植物。
