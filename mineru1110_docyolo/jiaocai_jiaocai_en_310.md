ZhejiangUniversity Theory of Computation, Fall-Winter 2015 Final Exam  

1.  $(20\%)$  Determine whether the following statements are true or false. If it is truewrite a  $\bigcirc$  otherwisea  $\times$  in the bracket before the statement.  

(a)（)Let  $L$   be any regular language. Then the number of equivalence classes respect to language  $L$  (i.e.  $x\approx_{L}y$  , if for all  $z\in\Sigma^{*}$   $x z\in L$  iff  $y z\in L$  ) is finite.   
(b)（） The language $\{a^{n}b^{n}w|w\in\{a,b\}^{*},n\in\mathbb{N}$ ,and $|w|=2n\}$ is context-free.  
(c)（） The language  $\{{}^{\ast}M^{\ast}{}^{\ast}w^{\ast}|M$  accepts  $w$   in less than 2016 steps  $\}$  is not recursive.   
(d)（） The language  $\{^{\Leftarrow}M^{\prime\prime}~|~M~$  is a TM and  $L(M)$  is Context-free but  $L(M)$  is not regular  $\}$  is not recursive.   
(e)（） The language $\{^{\ast}M_{1}^{\,,\ast}\,^{\ast}M_{2}^{\,,\ast}\,\mid M_{1}$ and $M_{2}$  are TMs, and $M_{1}$ halts on $e$ but $M_{2}$   doesn't halt on  $e\}$  is recursively enumerable, but not recursive.   
(f) ( ) The set of all primitive recursive functions is a proper subset of the set of all   $\mu$  -recursive functions.   
(g)  $()$  Let  $A$  and  $B$  be two disjoint recursively enumerable languages. If  $A\cup B$  is also be recursively enumerable, then it is possible that neither   $A$  nor  $B$  is decidable.   
(h)（)Let  $H_{10}=\{\cdots M^{\ast}|M$  is a TM and  $10\in L(M)\}$   and  $\tau_{\mathrm{1}}$  and  $\tau_{2}$  are recursive functions. If  $H_{10}\leq_{\tau_{1}}L$  and  $\overline{{H_{10}}}\leq_{\tau_{2}}L$  ,then  $L$   is recursive enumerable but not recursive.   
(i)（) Suppose SAT  $\le_{P}L$   and  $L\in\mathbb{P}$  . Then  $\mathbb{P}=\mathbb{N P}$    
(j)（)Let  $H=\{^{\cdots}M^{\ast\,\ast}w^{\ast}|$  TM  $M$   halts on input string  $w\}$  , then  $H$  is  $\mathbb{N P}$  -complete.
