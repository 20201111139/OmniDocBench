# 材料成分  

主料：香葱500g，大葱500g，紫葱头500g，切面（细）5kg;  
辅料：水 $300\,\mathrm{g}$ ，油菜500g，食用油500g;  
调料：酱油1kg。  

# 制作过程  

香葱切段，大葱、葱头切丝，油菜切开备用；葱油制作：锅内放油烧至三成热，将香葱、大葱、葱头入锅小火熬制20分钟后加酱油、水，开锅10后浇上葱油、撒上香葱粒即可（原料按35碗计算）。  

# 工艺技巧  

面条要细；熬油温度不宜太高。  

# 品质特点  

柔韧爽滑，葱香可口。
