# 32 飞  

从原有棋子出发，向“日”字形的对角线交点处下一子叫“飞”，也叫“小飞”。  

图2-116相对于图中原有的白棋一子来说，无论白方在A、B、C、D、E、F、G、H这 些字母代表的位置上任意一处行棋，都可以称其为飞。  

![](images/cecc0b31a0e6108df7dedbe8a59785a2e5ca006119ca16fe2f684ce6773ede6d.jpg)  
图2-116  

图2-117相对于白方原有的一子来说，白 $\triangle$ 是飞。反过来，相对于白方△一子，白原有的一子也叫作飞。  

飞的形式里还有“大飞”。它是指在原有棋子呈“目”字形的对角线交点处行棋。  

图2-118白△就叫白棋原有一子的大飞。同样，假若白方今后在A、B、C、D、E、F、G任何一处行棋，都叫大飞。
