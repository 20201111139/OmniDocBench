已知三角形的两边及其夹角，求作三角形

已知：线段 \( a \)，\( b \)，\( \angle \alpha \)，求作：\(\triangle ABC\)，使 \( BC = a \)，\( AB = c \)，\( \angle ABC = \angle \alpha \)

$\begin{aligned}& a \\& b\end{aligned}$

$\begin{aligned}& \text { 作法与示范 } \\& B \quad A \\& D' C \\& N \\& M\end{aligned}$

$\begin{aligned}& \text { 作法 } \\& (1) \text { 作 } \angle MBN = \angle \alpha \\& (2) \text { 在射线 } BM \text { 上截取 } BC = a, \text { 在射线 } BN \text { 上截取 } BA = b \\& (3) \text { 连接 } AC \\& \text { 则 } \triangle ABC \text { 为所求作的三角形 }\end{aligned}$
