设
$$f(x)=\sum_{n=1}^{\infty} \frac{(-1)^{n}}{2 n+1} x^{2 n+1}, x \in[-1,1] .$$

由幂级数相关性质可知
$$f^{\prime}(x)=\sum_{n=1}^{\infty}(-1)^{n} x^{2 n}=-\frac{x^{2}}{1+x^{2}}=\frac{1}{1+x^{2}}-1, x \in(-1,1) .$$

即 $f(x)=\arctan x-x$ ，故
$$\sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n(2 n+1)}=\ln 2+2 f(1)=\ln 2+2\left(\frac{\pi}{4}-1\right)=\ln 2+\frac{\pi}{2}-2$$

4. 设 $f(x, y)=x^{3}-3 x^{2} y-y^{3}+x^{2}-y$. (1) 证明: 存在 $\delta>0$ ，以及定义于 $(-\delta, \delta)$ 上的连续可微函数 $y=y(x)$, 满足 $y(0)=0$, 以及 $f(x, y(x))=0$. (2) 证明: $x=0$ 时 (1) 中的 $y(x)$ 取到极小值.
[答案]: 证明 (1) 注意到 $f(0,0)=1, \frac{\partial f}{\partial y}(0,0)=-3 x^{2}-3 y^{2}-\left.1\right|_{(0,0)}=-1 \neq 0$. (2) 在 $x^{3}-3 x^{2} y-y^{3}+x^{2}-y=$ 0 两边同时对 $x$ 求导，有
$$3 x^{2}-6 x y-3 x^{2} y^{\prime}-3 y^{2} y^{\prime}+2 x-y^{\prime}=0 .$$

代入 $x=0$ 和 $y(0)=0$, 得到 $y^{\prime}(0)=0$ 。重复上述操作, 得到
$$6 x-6 y-12 x y^{\prime}-3 x^{2} y^{\prime \prime}-6 y\left(y^{\prime}\right)^{2}-3 y^{2} y^{\prime \prime}+2-y^{\prime \prime}=0 .$$

代入 $x=0$ 和 $y(0)=y^{\prime}(0)=0$ ，得到 $y^{\prime}(0)=2>0$ 。 [解析]:
5. 设 $f(x)$ 在 $(-\infty,+\infty)$ 上可导且下凸, 证明：对任意的实数 $x$ ，都有 $f\left(x+f^{\prime}(x)\right) \geqslant f(x)$.
[答案]: 证明由题意可知对任意实数 $x, y$, 有
$$f(y) \geqslant f(x)+f^{\prime}(x)(y-x)$$

故 $f\left(x+f^{\prime}(x)\right) \geqslant f(x)+f^{\prime}(x)\left(\left(x+f^{\prime}(x)\right)-x\right)=f(x)+\left(f^{\prime}(x)\right)^{2} \geqslant f(x)$ [解析]:
6. 设 $x_{n}=\sum_{k=0}^{n} \frac{1}{k!}, n=1,2, \cdots$, 求极限 $\lim _{n \rightarrow \infty}\left(\frac{\ln x_{n}}{\sqrt[n]{\mathrm{e}}-1}-n\right)$.
[答案]: 解注意到
$$x_{n}=\mathrm{e}-\sum_{k=n+1}^{\infty} \frac{1}{k!} \stackrel{\text { def }}{=} \mathrm{e}-r_{n}, \lim _{n \rightarrow \infty} r_{n}=0$$

又
$$\sqrt[n]{\mathrm{e}}-1=\frac{1}{n}+\frac{1}{2 n^{2}}+o\left(\frac{1}{n^{2}}\right)$$

故
$$\text { 原式 }=\lim _{n \rightarrow \infty} n\left(\ln x_{n}-1-\frac{1}{2 n}\right)=\lim _{n \rightarrow \infty} n\left(\ln \frac{\mathrm{e}-r_{n}}{\mathrm{e}}-\frac{1}{2 n}\right)=-\frac{1}{2}$$
