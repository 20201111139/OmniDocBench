7.7 几 何 形式
(4) 核心笔记
(1) 复 数 $z=a+b i \xrightarrow{- \text {-对应 }}$ 点 $Z(a$, $b) \xrightarrow{\text { 一一对应 }}$ 向量 $\overrightarrow{O Z}=(a, b)$ 。
（2）" $\cdot$ i"的几何意义：逆时针旋转 $90^{\circ}$ 。
例 如, $3+2 \mathrm{i} \xrightarrow{\text { 逆时针旋转 } 90^{\circ}}(3+2 \mathrm{i})$. $\mathrm{i}=-2+3 \mathrm{i}$ 。
$$Z(3,2) \xrightarrow{\text { 逆时针旋转 } 90^{\circ}} Z^{\prime}(-2,3)$$
（3）" $\div \mathrm{i} "$ 的几何意义：顺时针旋转 $90^{\circ}$ 。
例如, $3+2 \mathrm{i} \xrightarrow{\text { 顺时针旋转 } 90^{\circ}} \frac{3+2 \mathrm{i}}{\mathrm{i}}=2-3 \mathrm{i}$ 。
$$Z(3,2) \xrightarrow{\text { 顺时针旋转 } 90^{\circ}} Z^{\prime}(2,-3)$$
（4）" $\bar{z}$ "的几何意义：与 $z$ 关于 $x$ 轴对称。
