数学建模
- $\Delta=0 \mathbf{A}$, 有两个相同的特征值
$$\begin{array}{c}
\frac{d \mathbf{u}}{d t}=\mathbf{P}^{-1} \mathbf{A P u} \\
\alpha=\operatorname{tr}(\mathbf{A})=\lambda_{1}+\lambda_{2} \\
\beta=|\mathbf{A}|=\lambda_{1} \lambda_{2}
\end{array}$$
$$\left\{\begin{array} { l } 
{ \frac { d u } { d t } = \lambda u } \\
{ \frac { d v } { d t } = \lambda v }
\end{array} \quad \left\{\begin{array}{l}
u=u_{0} e^{\lambda t} \\
v=v_{0} e^{\lambda t}
\end{array} \quad \frac{u}{v}=\frac{u_{0}}{v_{0}}\right.\right.$$
- $\alpha<0, \lambda<0$
- $\alpha>0, \lambda>0$
