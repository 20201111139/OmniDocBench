\text { if } S_{\triangle A D E}=4 x . \quad S_{\triangle A B C}=9 x \\

\therefore \frac{S_{\triangle A D E}}{S_{\triangle D \text { CIDBCE }}}=\frac{4 x}{5 x}=\frac{4}{5} \text {. }
\end{array}$$
