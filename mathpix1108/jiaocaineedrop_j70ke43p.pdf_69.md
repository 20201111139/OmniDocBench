1. 三角恒等变换

万能置换公式: $\sin x=\frac{2 \tan \frac{x}{2}}{1+\tan ^{2} \frac{x}{2}} ; \cos x=\frac{1-\tan ^{2} \frac{x}{2}}{1+\tan ^{2} \frac{x}{2}} ; \quad \tan x=\frac{2 \tan \frac{x}{2}}{1-\tan ^{2} \frac{x}{2}}$ ；
(1) 积化和差公式
$$\begin{array}{l}
\sin A \cos B=\frac{1}{2}[\sin (A+B)+\sin (A-B)] ; \cos A \sin B=\frac{1}{2}[\sin (A+B)-\sin (A-B)] ; \\
\cos A \cos B=\frac{1}{2}[\cos (A+B)+\cos (A-B)] ; \sin A \sin B=\frac{1}{2}[\cos (A-B)-\cos (A+B)] ;
\end{array}$$
(2) 和差化积公式
$$\begin{array}{l}
\sin A+\sin B=2 \sin \frac{A+B}{2} \cos \frac{A-B}{2} ; \sin A-\sin B=2 \cos \frac{A+B}{2} \sin \frac{A-B}{2} ; \\
\cos A+\cos B=2 \cos \frac{A+B}{2} \cos \frac{A-B}{2} ; \cos A-\cos B=-2 \sin \frac{A+B}{2} \sin \frac{A-B}{2} ;
\end{array}$$
