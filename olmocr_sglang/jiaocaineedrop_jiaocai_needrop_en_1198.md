例 26.2022 联考·全国·南京外国语学校模拟预测) 已知抛物线 \(C: x^2 = 4y, F\) 为其焦点，过 \(F\) 的直线 \(l\) 与 \(C\) 交于不同的两点 \(A, B, B\).

(1) 若直线 \(l\) 斜率为 3，求 \(|AB|\);

(2) 如图，\(C\) 在点 \(A\) 处的切线与在点 \(B\) 处的切线交于点 \(P\)，连接 \(PF\)，证明：\(|PF|^2 = |AF| \cdot |BF|\).

【解析】(1) 由抛物线 \(C: x^2 = 4y, F(0, 1)\)，

若直线 \(l\) 的斜率为 3，则直线 \(l\) 的方程为 \(y = 3x + 1\).

设 \(A(x_1, y_1), B(x_2, y_2)\)，

由 \(\begin{cases} y = 3x + 1 \\ x^2 = 4y \end{cases}\) 消去 \(x\) 得 \(x^2 - 12x - 4 = 0\)，所以 \(\Delta > 0, x_1 + x_2 = 12,\)

所以 \(|AB| = |AF| + |FB| = y_1 + 1 + y_2 + 1 = 3(x_1 + x_2) + 4 = 40.\)

(2) 证明：由题可知，直线 \(l\) 的斜率存在，设直线 \(l\) 的方程为 \(y = kx + 1, A(x_1, y_1), B(x_2, y_2)\)，

抛物线方程为 \(x^2 = 4y, y' = \frac{x^2}{4}, y = \frac{x^2}{4}\).

所以以 \(A, B\) 为切点的切线方程分别为 \(x_1x = 2y + 2y_1, x_2x = 2y + 2y_2,\)

由 \(\begin{cases} x_1 = \frac{2y_1}{x_2} \\ x_2 = \frac{2y_2}{x_1} \end{cases}\) 消去 \(y\) 得 \(x_1^2 - 4kx_1 - 4 = 0\)，所以 \(\Delta > 0, x_1 + x_2 = 4k, x_1x_2 = -4.\)

这两条切线的斜率分别为 \(k_1 = \frac{x_1}{2}, k_2 = \frac{x_2}{2}\).

由 \(k_1k_2 = \frac{x_1x_2}{4} = -\frac{1}{4} = -1\)，所以 \(PA \perp PB\).

设 \(P(x_0, y_0)\)，则 \(x_1x_2 = 2y_1 + 2y_2, \) 所以 \(x_1 = \frac{2(y_1 - y_0)}{x_1 - x_2}, y_1 = \frac{x_1x_0 - y_1}{2}, y_2 = \frac{x_1x_0 - y_2}{2}, k_{AB} = \frac{x_0}{2} = -1, k_{FF} = \frac{x_0}{2} = -1,\)

当 \(k = 0\) 时，则 \(x_0 = 0, k_{AB} = \frac{x_0}{2}, k_{FF} = \frac{x_0}{2},\) 所以 \(k_{AB}, k_{FF} = \frac{x_0}{2}, \) 所以 \(k_{AB}, k_{FF} = \frac{x_0}{2},\) 所以 \(|PF|^2 = |AF| \cdot |BF|\).
