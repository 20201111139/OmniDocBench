### (三) 切换音频方式

终端的音频类型包括麦克风和扬声器，在使用麦克风和扬声器的过程中，若终端成功外接音频附件后，可以根据需要切换音频方式。具体说明和操作方法见表 6-7-8。

| 音频方式 | 具体说明 | 操作方法 |
|----------|----------|----------|
| 跟随 PTT | 使用终端自身的“PTT”键进行语音发射则启用内部麦克风，使用外接的音频附件进行语音发射则启用外部麦克风 | 按“麦克风设置切换”快捷键，快速切换麦克风的音频方式。按“喇叭设置切换”快捷键，快速切换扬声器的音频方式。 |
| 仅本机 | 仅使用终端自身的麦克风和扬声器 |  |
| 仅外设 | 仅使用外接音频附件的麦克风和扬声器，如耳机、扬声器话筒 |  |
| 外设优先 | 终端外接音频附件时使用外部麦克风和扬声器，无外接音频附件时使用终端自身的麦克风和扬声器 |  |
