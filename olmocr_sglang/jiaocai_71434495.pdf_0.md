1. 求定积分 \( \int_{0}^{\frac{\pi}{2}} \frac{\sin x \cos x}{\sin^4 x + \cos^4 x} \, dx \).

【答案】令 \( t = \sin^2 x \)，则 \( dt = 2 \sin x \cos x \, dx \)，\( \sin^4 x + \cos^4 x = 2t^2 - 2t + 1 \)，则

\[ \int_{0}^{\frac{\pi}{2}} \frac{\sin x \cos x}{\sin^4 x + \cos^4 x} \, dx \, dt = \int_{0}^{1} \frac{1}{2t^2 - 2t + 1} \, dt = \frac{1}{12} \int_{0}^{1} \left( \frac{1}{t-2} - \frac{1}{t+1} \right) \, dt = \frac{\ln 2}{12}. \]

【解析】

2. 计算二重积分 \( \iint_D \sin \left( \sqrt{x^2 + y^2} \right) \, dx \, dy \)，其中区域 \( D = \{(x, y) \mid 0 \leq x, y \leq \sqrt{x} \} \).

【答案】不考虑 \( D = \{(x, y) \mid 0 \leq x, y \leq \sqrt{x} \} \)，由对称性可知

\[ \iiint_D \sin \left( \sqrt{x^2 + y^2} \right) \, dx \, dy = 2 \int_{0}^{\frac{\pi}{2}} \sin x^2 \, dx \, dy = 2 \int_{0}^{\frac{\pi}{2}} \sin x^2 \, dx = 2 \]

【解析】

3. 求级数 \( \sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n(2n+1)} \) 的和。

【答案】考虑

\[ \sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n(2n+1)} = \sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n} + \sum_{n=1}^{\infty} \frac{(-1)^{n}}{2n+1}. \]

注意到

\[ \ln(1+x) = \sum_{n=1}^{\infty} (-1)^{n-1} \frac{x^n}{n}, x \in (-1, 1). \]

设

\[ f(x) = \sum_{n=1}^{\infty} \frac{(-1)^{n}}{n} x^{2n+1}, x \in [-1, 1]. \]

由幂级数相关性质可知

\[ f'(x) = \sum_{n=1}^{\infty} (-1)^{n} x^{2n} = -\frac{x^2}{1+x^2} = -\frac{1}{1+x^2} - 1, x \in (-1, 1). \]

即 \( f(x) = \arctan x - x \)，故

\[ \sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n(2n+1)} = \ln 2 + 2 f(1) = \ln 2 + 2 \left( \frac{\pi}{4} - 1 \right) = \ln 2 + \frac{\pi}{2} - \frac{3}{2}. \]

【解析】

4. 设 \( f(x, y) = x^3 - 3x^2y - y^2 + x^2 - y \)。（1）证明：存在 \( \delta > 0 \)，以及定义于 \( (-\delta, \delta) \) 上的连续可微函数 \( y = y(x) \) 满足 \( y(0) = 0 \)，以及 \( f(x, y(x)) = 0 \)。（2）证明：\( x = 0 \) 时（1）中 \( y(x) \) 取极小值。

【答案】（1）注意到 \( f(0, 0) = 1, \frac{\partial f}{\partial y}(0, 0) = -3x^2 - 3y^2 - 11_{1|y(0)} = -1 \neq 0 \)。（2）在 \( x^3 - 3x^2y - y^2 + x^2 - y = 0 \) 两边同时对 \( x \) 求导，有

\[ 3x^2 - 6xy - 3x^2y' - 2y + y' = 0. \]

代入 \( x = 0 \) 和 \( y(0) = 0 \)，得到 \( y'(0) = 0 \)。重复上述操作，得到

\[ 6x - 6y - 12xy' - 6y(y')^2 - 3y^2 + x^2 + y'' = 0. \]

代入 \( x = 0 \) 和 \( y(0) = 0 \)，得到 \( y''(0) = 2 \)。【解析】

5. 设 \( f(x) \) 在 \( (-\infty, \infty) \) 上连续且有定义，证明：对任意的实数 \( x, y \)，都有 \( f(x + f'(x)) > f(x) \)。

【答案】证明由题意可知对任意实数 \( x, y \)，有

\[ f(y) > f(x) + f'(x)(y-x), \]

故 \( f(x + f'(x)) > f(x) + f'(x)(x + f'(x) - x) = f(x) + (f'(x))^2 > f(x) \)。【解析】

6. 设 \( x_{n+1} = \sum_{k=1}^{n} \frac{1}{k!} x_{k}, n = 1, 2, \ldots \)，求极限 \( \lim_{n \to \infty} \left( \frac{\ln x_{n}}{\sqrt{e} - 1} - n \right) \).

【答案】注意到

\[ x_{n} = e - \sum_{k=n+1}^{\infty} \frac{1}{k!} \approx e - r_{n}, \lim_{n \to \infty} r_{n} = 0. \]

又

\[ \sqrt{e} - 1 = \frac{1}{2} + \frac{1}{2 \sqrt{e}} + o \left( \frac{1}{n^{3}} \right) \]

故

\[ \lim_{n \to \infty} n \left( \ln x_{n} - 1 - \frac{1}{2n} \right) = \lim_{n \to \infty} n \left( \ln \frac{e - r_{n}}{e} - \frac{1}{2n} \right) = -\frac{1}{2} \]
